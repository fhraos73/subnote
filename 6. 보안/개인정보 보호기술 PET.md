#정보관리기술사 #보안 #개인정보 #PET #프라이버시 #정리

## 정의
프라이버시 강화 기술, PET(Privacy Enhancing Technology)
- 개인정보의 수집, 저장, 처리, 공유 과정에서 프라이버시를 보호하기 위한 기술적 수단의 총칭
- 데이터 활용과 개인정보 보호를 동시에 달성하는 기술

## 키워드
* 동형암호, 차분 프라이버시, 연합학습, 영지식 증명, 가명화, 익명화

## 암기법
* 동차연영가익: (PET 기술) 동형암호, 차분 프라이버시, 연합학습, 영지식 증명, 가명화, 익명화

정책 협상기술 P3P(W3C 제정 개인정보보호 정책 표준규격)
암호 기술 DES/RSA/ECC, SSL/VPN
필터링 기술 방화벽/IDS/IPS/NAC, 안티바이러스/캡쳐방지, 피싱 모니터링
익명화 기술 Mix-net(클라이언스 서버 익명성 제공), ISO25237, ISO20889
인증 기술 PKI, i-PIN, SSO, Open ID


## 특징
- 데이터 활용: 프라이버시 보호하면서 데이터 분석 가능
- 기술적 보호: 법적 보호를 기술로 구현
- 다양한 기법: 암호화, 익명화, 분산 처리 등
- 규제 대응: GDPR, 개인정보보호법 준수 지원

## 목적
- 개인정보 보호와 데이터 활용의 균형
- 프라이버시 침해 위험 최소화
- 법적 규제 준수 지원

## 구성요소
| 구분 | 기술 | 설명 |
|------|------|------|
| 암호화 | 동형암호 | 암호화 상태에서 연산 |
| 암호화 | 안전한 다자간 연산 | 데이터 공개 없이 공동 연산 |
| 익명화 | 가명화 | 식별자를 가명으로 대체 |
| 익명화 | 익명화 | 재식별 불가능하게 처리 |
| 익명화 | k-익명성 | k명 이상 동일 속성 |
| 분산 | 연합학습 | 데이터 이동 없이 학습 |
| 분산 | 차분 프라이버시 | 노이즈 추가로 개인 보호 |
| 검증 | 영지식 증명 | 정보 공개 없이 검증 |
