#정보관리기술사 #보안 #양자암호 #하이브리드 #옵션 #정리

## 정의
QKD와 PQC를 결합한 이중 보안 통신, 하이브리드 양자암호통신
- 양자 키 분배(QKD) 기술과 양자내성 암호(PQC) 기술을 동시에 결합하여 보안을 강화하는 통신 방법
- QKD의 거리 제한과 PQC의 수학적 가정 의존성을 상호 보완

## 키워드
* QKD, PQC, 이중 보안, 양자 채널, 키 캡슐화, 하이브리드 키

## 암기법
* QP이: (구성) QKD + PQC = 이중 보안

## 특징
- 이중 보안: QKD와 PQC 동시 적용으로 보안 강화
- 상호 보완: QKD 거리 제한 + PQC 수학적 가정 보완
- 점진적 전환: 기존 인프라에서 양자 보안으로 전환 용이
- 유연성: 환경에 따라 QKD 또는 PQC 비중 조절

## 목적
- 양자암호통신의 실용성 향상
- 단일 기술 의존 위험 분산
- 양자 컴퓨터 시대 대비 보안 체계 구축

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| QKD | 양자 키 분배 | 물리적 보안, 도청 감지 가능 |
| QKD | 양자 채널 | 광섬유/자유공간 통한 키 전송 |
| PQC | 양자내성암호 | 수학적 보안, 거리 제한 없음 |
| PQC | 키 캡슐화 | CRYSTALS-Kyber 등 |
| 결합 | 하이브리드 키 | QKD 키 + PQC 키 결합 |
| 결합 | 키 결합 함수 | XOR, KDF 등으로 최종 키 생성 |

## 구성도
```
[송신자]                              [수신자]
    │                                     │
    ├──[QKD 채널]──────────────────────────┤
    │  (양자 키 K1 분배)                    │
    │                                     │
    ├──[PQC 채널]──────────────────────────┤
    │  (양자내성 키 K2 교환)                 │
    │                                     │
    └──────────────┬──────────────────────┘
                   │
            [키 결합 함수]
            K = KDF(K1 || K2)
                   │
            [하이브리드 키 K]
                   │
            [대칭키 암호화]
            (AES 등)
```
