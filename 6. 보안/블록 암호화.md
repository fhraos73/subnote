#정보관리기술사 #보안 #암호화 #블록암호 #정리

## 정의
고정 크기 블록 단위 데이터 암호화 기술, 블록 암호화
- 평문을 고정된 크기의 블록으로 나누어 각 블록을 독립적으로 암호화하는 대칭키 암호화 방식
- Feistel 구조 또는 SPN 구조를 기반으로 혼돈(Confusion)과 확산(Diffusion) 원리 적용

## 키워드
* Feistel, SPN, S-Box, P-Box, ECB, CBC, CTR, DES, AES, SEED, ARIA

## 암기법
* 혼대치수 확전순P: (암호화 원리) 혼돈-대치-치환-S-BOX / 확산-전치-순열-P-BOX
* 운모: ECB CBC CFB OFB CTR (운영모드)
* 피스: Feistel(DES,SEED), SPN(AES,ARIA)

## 특징
- 고정 블록 크기: 64bit, 128bit 등 고정 크기 단위 처리
- 라운드 반복: 여러 라운드를 거쳐 보안 강도 향상
- 운영 모드: ECB, CBC, CTR 등 다양한 모드 지원
- 패딩: 블록 크기에 맞지 않는 데이터 처리

## 목적
- 대용량 데이터의 효율적 암호화
- 높은 보안 강도 제공
- 다양한 운영 모드를 통한 유연한 적용

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 구조 | Feistel | 좌우 분리, XOR, Swap, Round 반복 (DES, SEED) |
| 구조 | SPN | S-Box, P-Box 이용 병렬 처리 (AES, ARIA) |
| 연산 | S-Box | 비선형 치환, 혼돈(Confusion) 제공 |
| 연산 | P-Box | 순열 변환, 확산(Diffusion) 제공 |
| 모드 | ECB | Electronic CodeBook, 사용 금지 권고 |
| 모드 | CBC | Cipher Block Chaining, 권장 모드 |
| 모드 | CTR | Counter 모드, 병렬 처리 가능, 권장 |

## 구성도
```
[평문 블록] ──▶ [라운드 함수] ──▶ [암호문 블록]
                    │
              [라운드 키]
                    │
     ┌──────────────┼──────────────┐
     ▼              ▼              ▼
[Feistel 구조]  [SPN 구조]    [운영 모드]
     │              │              │
 ┌───┴───┐      ┌───┴───┐      ┌───┴───┐
 ▼       ▼      ▼       ▼      ▼       ▼
[좌블록][우블록][S-Box][P-Box][ECB][CBC][CTR]
     │              │
   [XOR]        [혼돈+확산]
     │              │
  [Swap]       [병렬처리]
```
