#정보관리기술사 #보안 #양자암호 #QKD #정리 #보안/암호화

## 정의
양자역학 원리 기반 차세대 암호 기술, 양자암호기술
- 양자 중첩, 얽힘 등 양자물리학 특성을 정보통신기술에 적용하여 정보통신 인프라를 보호하는 차세대 암호 기술
- 양자 키 분배(QKD)를 통해 도청 불가능한 안전한 키 교환 실현

## 키워드
* QKD, BB84, 양자중첩, 양자얽힘, 편광, 단일광자, 큐비트(Qubit)

## 암기법
* 편위채퍼B: (기술요소) 편광, 위상, 양자채널, 퍼블릭채널, BB84프로토콜
* 양광단난프: (구현기술) 양자광원, 단일광자 검출기, 난수 발생기, 양자 암호 프로토콜

## 특징
- 도청 감지: 양자 상태 측정 시 변화 발생으로 도청 탐지
- 무조건적 안전성: 물리 법칙에 기반한 보안
- 양자 중첩: 0과 1 상태 동시 존재
- 양자 얽힘: 두 입자 간 상관관계

## 목적
- 양자 컴퓨터 시대의 안전한 암호 통신
- 도청 불가능한 키 분배
- 국가 핵심 인프라 보안 강화

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 기본 단위 | 큐비트(Qubit) | 0과 1 상태를 동시에 갖는 양자 비트 |
| 프로토콜 | BB84 | 편광 기반 양자 키 분배 프로토콜 |
| 채널 | 양자 채널 | 광섬유/자유공간 통한 양자 상태 전송 |
| 채널 | 퍼블릭 채널 | 기저 비교 등 고전 정보 교환 |
| 장비 | 양자광원 | 단일 광자 생성 장치 |
| 장비 | 단일광자 검출기 | 광자 수신 및 측정 |
| 장비 | 난수 발생기 | 양자 난수 생성 |

## 구성도
```
[송신자 Alice]                    [수신자 Bob]
      │                                │
[양자광원] ──▶ [양자 채널] ──▶ [단일광자 검출기]
(단일광자 생성)  (광섬유/자유공간)   (광자 측정)
      │                                │
[편광 인코딩]                    [편광 측정]
(기저 선택)                      (기저 선택)
      │                                │
      └────────[퍼블릭 채널]────────────┘
              (기저 비교, 키 추출)
                     │
              [도청 검출]
              (오류율 확인)
                     │
              [비밀키 생성]
```
