#정보관리기술사 #보안 #딥러닝 #AI보안 #적대적공격 #정리

## 정의
딥러닝 모델 보안 취약점, 딥러닝 취약점
- 딥러닝/머신러닝 모델의 학습, 추론 과정에서 발생할 수 있는 보안 취약점
- 적대적 공격, 데이터 포이즈닝, 모델 추출 등 AI 고유의 보안 위협

## 키워드
* 적대적 예제, 데이터 포이즈닝, 모델 추출, 멤버십 추론, 백도어 공격

## 암기법
* 적모멤 대백모: 적대적 공격, 모델 추출, 멤버쉽 추론, 데이터 포이즈닝, 백도어 공격, 모델 역전

## 특징
- 블랙박스 특성: 모델 내부 동작 불투명
- 데이터 의존: 학습 데이터 품질이 보안에 영향
- 전이 가능성: 한 모델의 공격이 다른 모델에도 유효
- 탐지 어려움: 미세한 변조로 오분류 유발

## 목적
- AI/ML 시스템 보안 위협 식별
- 안전한 AI 모델 개발 및 운영
- AI 신뢰성 확보

## 구성요소
| 구분 | 취약점 | 설명 |
|------|--------|------|
| 추론 | 적대적 예제 | 미세 변조로 오분류 유발 |
| 추론 | 모델 추출 | 쿼리를 통한 모델 복제 |
| 추론 | 멤버십 추론 | 학습 데이터 포함 여부 추론 |
| 학습 | 데이터 포이즈닝 | 학습 데이터 오염 |
| 학습 | 백도어 공격 | 트리거 패턴 삽입 |
| 학습 | 모델 역전 | 학습 데이터 복원 |
| 프라이버시 | 속성 추론 | 민감 속성 추론 |
| 프라이버시 | 학습 데이터 유출 | 모델에서 데이터 추출 |
