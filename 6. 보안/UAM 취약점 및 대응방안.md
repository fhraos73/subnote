#정보관리기술사 #보안 #UAM #도심항공 #정리

## 정의
도심항공모빌리티 보안 위협 및 대응, UAM 취약점
- UAM(Urban Air Mobility) 시스템의 통신, 제어, 운항 관리에서 발생할 수 있는 보안 취약점과 대응 방안
- 항공기, 버티포트, 운항관리시스템(UTM) 등 UAM 생태계 전반의 사이버 보안

## 키워드
* eVTOL, UTM, 버티포트, GPS 스푸핑, 통신 보안, 항공 사이버 보안

## 암기법
* 항버운통: (구성요소) 항공기(eVTOL), 버티포트, 운항관리(UTM), 통신

## 특징
- 고위험 시스템: 사이버 공격이 인명 피해로 직결
- 복잡한 생태계: 다양한 이해관계자, 시스템 연동
- 실시간 요구: 지연 없는 보안 처리 필요
- 신규 영역: 보안 표준 및 규제 미성숙

## 목적
- UAM 시스템 보안 위협 식별
- 안전한 도심항공 서비스 운영
- 항공 사이버 보안 체계 구축

## 구성요소
| 구분 | 취약점 | 대응방안 |
|------|--------|----------|
| 항공기 | GPS 스푸핑 | 다중 항법, 관성항법 |
| 항공기 | 통신 가로채기 | 암호화, 인증 |
| 항공기 | 제어 시스템 취약점 | 보안 설계, 펌웨어 서명 |
| 버티포트 | 물리적 보안 | 접근제어, CCTV |
| 버티포트 | 충전 인프라 취약점 | 인증, 모니터링 |
| UTM | 운항 데이터 조작 | 무결성 검증, 이중화 |
| UTM | 서비스 거부 | DDoS 방어, 이중화 |
| 통신 | C2 링크 취약점 | 암호화, 주파수 호핑 |
| 통신 | 데이터 링크 취약점 | 인증, 무결성 검증 |
