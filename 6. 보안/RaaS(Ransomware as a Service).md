#정보관리기술사 #보안 #랜섬웨어 #RaaS #사이버범죄 #정리 #보안/공격/위협

## 정의
서비스형 랜섬웨어, RaaS(Ransomware as a Service)
- 다크웹과 같은 익명 네트워크를 이용하여 비용만 지급하면 랜섬웨어 공격을 할 수 있게끔 서비스 형태로 제공되는 랜섬웨어
- 기술적 지식 없이도 랜섬웨어 공격이 가능한 범죄 서비스 모델

## 키워드
* 다크웹, 랜섬웨어, 수익 분배, 어필리에이트, 이중 협박, 암호화폐

## 암기법
* 개운공수: (RaaS 구조) 개발자, 운영자, 공격자(어필리에이트), 수익 분배

## 특징
- 진입 장벽 낮음: 기술 없이 공격 가능
- 수익 분배: 개발자와 공격자 간 수익 분배
- 지속적 업데이트: 탐지 회피 기술 지속 개선
- 이중 협박: 암호화 + 데이터 유출 협박

## 목적
- 랜섬웨어 공격의 대중화
- 사이버 범죄 수익 극대화
- 공격자 익명성 보장

## 구성요소
| 구분  | 구성요소    | 설명                     |
| --- | ------- | ---------------------- |
| 참여자 | 개발자     | 랜섬웨어 개발 및 유지보수         |
| 참여자 | 운영자     | 인프라 운영, 협상             |
| 참여자 | 어필리에이트  | 실제 공격 수행자              |
| 서비스 | 랜섬웨어 키트 | 공격 도구 제공               |
| 서비스 | 관리 패널   | 감염 현황, 수익 관리           |
| 서비스 | 기술 지원   | 공격 방법 안내               |
| 수익  | 수익 분배   | 개발자 20-30%, 공격자 70-80% |
| 결제  | 암호화폐    | 비트코인, 모네로 등            |

## 구성도
```
[RaaS 생태계]

[개발자/운영자]
      │
[랜섬웨어 개발]
[인프라 구축]
[다크웹 마켓]
      │
      ▼
[RaaS 플랫폼]
      │
    ┌─┴─┬─────┬─────┐
    ▼   ▼     ▼     ▼
[랜섬웨어][관리패널][기술지원][협상서비스]
    │
    ▼
[어필리에이트 (공격자)]
      │
[피싱/취약점 공격]
      │
      ▼
[피해 조직]
      │
[암호화 + 데이터 유출 협박]
      │
      ▼
[몸값 지불 (암호화폐)]
      │
      ▼
[수익 분배]
(개발자 20-30% / 공격자 70-80%)

[주요 RaaS 그룹]
    │
    ├── LockBit
    ├── BlackCat (ALPHV)
    ├── Conti
    └── REvil (Sodinokibi)

[대응 방안]
    │
    ├── 백업 및 복구 체계
    ├── 보안 인식 교육
    ├── 취약점 관리
    ├── 네트워크 세그멘테이션
    └── 침해사고 대응 계획
```
