#정보관리기술사 #보안 #암호화 #ECC #옵션 #정리

## 정의
타원곡선 이산대수 문제 기반 공개키 암호화, ECC
- 타원곡선의 수학적 특성을 이용하여 비대칭키를 생성하는 암호화 알고리즘
- RSA 대비 짧은 키 길이로 동등한 보안 강도 제공

## 키워드
* 타원곡선, 이산대수, ECDSA, ECDH, 공개키, 개인키, 경량암호

## 암기법
* 타이구일: (특징) 타원곡선, 이산대수, 구현용이, 일방향성

## 특징
- 짧은 키 길이: RSA 2048bit ≈ ECC 224bit 동등 보안
- 경량성: 적은 메모리, 저전력 환경 적합
- 구현 용이: IC카드, PDA, IoT 디바이스 적용
- 일방향성: 타원곡선 이산대수 문제의 어려움 기반

## 목적
- RSA의 속도와 키 길이 문제 해결
- IC카드, PDA 등 제한된 환경의 암호화
- 모바일, IoT 환경의 경량 암호화

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 수학적 기반 | 타원곡선 | y² = x³ + ax + b (mod p) |
| 수학적 기반 | 이산대수 | 타원곡선 상의 점 연산 |
| 알고리즘 | ECDSA | 전자서명 알고리즘 |
| 알고리즘 | ECDH | 키 교환 알고리즘 |
| 키 | 개인키 | 무작위 정수 d |
| 키 | 공개키 | Q = d × G (G: 기준점) |
