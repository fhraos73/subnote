#정보관리기술사 #보안 #포렌식 #안티포렌식 #옵션 #정리

## 정의
디지털 포렌식 방해 기술, 안티포렌식(Anti-Forensics)
- 디지털 포렌식 수사를 방해하거나 증거를 은닉, 파괴, 위조하여 수사를 어렵게 만드는 기술 및 행위
- 데이터 영구 삭제, 암호화, 은닉, 물리적 파손 등 다양한 기법 포함

## 키워드
* 데이터 와이핑, 암호화, 스테가노그래피, 슬랙 공간, 타임스탬프 조작

## 암기법
* 삭암파은: (안티포렌식 기법) 데이터 삭제, 암호화, 물리적 파손, 데이터 은닉

## 특징
- 증거 인멸: 디지털 증거 파괴 또는 은닉
- 수사 방해: 포렌식 분석 시간 증가
- 다양한 기법: 소프트웨어/하드웨어 기법 혼용
- 대응 필요: 안티-안티포렌식 기술 발전

## 목적
- 디지털 증거 인멸
- 포렌식 수사 방해 및 지연
- 범죄 은폐

## 구성요소
| 구분 | 기법 | 설명 |
|------|------|------|
| 데이터 삭제 | Disk Wipe | 모든 섹터 덮어쓰기 (Null, Random) |
| 데이터 삭제 | Degausser | 강력한 자기장으로 데이터 파괴 |
| 데이터 삭제 | 덮어쓰기 | 삭제 후 여러 번 덮어쓰기 |
| 암호화 | 파일 암호화 | ZIP, RAR 암호화 |
| 암호화 | 디스크 암호화 | BitLocker, VeraCrypt |
| 암호화 | 스테가노그래피 | 이미지/오디오에 데이터 은닉 |
| 물리적 파손 | 소각/파쇄 | 저장매체 물리적 파괴 |
| 물리적 파손 | 전자파 | EMP로 데이터 파괴 |
| 데이터 은닉 | 슬랙 공간 | 파일 슬랙에 데이터 숨김 |
| 데이터 은닉 | ADS | NTFS 대체 데이터 스트림 |
| 조작 | 타임스탬프 | 파일 시간 정보 조작 |
| 조작 | 로그 삭제 | 시스템/이벤트 로그 삭제 |

```
         │

[데이터 삭제] [암호화]     [물리적 파손] [데이터 은닉]
    │         │            │            │
    ▼         ▼            ▼            ▼

[덮어쓰기]  [스테가노]    [물리손상]   [타임스탬프]

[대응 방안]
    │
    ├── 데이터 카빙 (삭제 파일 복구)
    ├── 암호 해독 (브루트포스, 사전공격)
    ├── 메타데이터 분석
    └── 슬랙 공간 분석
```
