#정보관리기술사 #보안 #스마트팩토리 #OT보안 #옵션 #정리

## 정의
스마트팩토리 환경 보안 위협 및 대응, 스마트팩토리 취약점
- 제조 환경의 IT/OT 융합으로 발생하는 보안 취약점과 이에 대한 대응 방안
- SCADA, PLC, ICS 등 산업제어시스템의 사이버 보안 위협

## 키워드
* OT 보안, SCADA, PLC, ICS, 스턱스넷, IT/OT 융합, IEC 62443

## 암기법
* IT OT 융합: (특징) IT 보안 + OT 보안 통합 필요

## 특징
- IT/OT 융합: 기존 폐쇄망의 외부 연결 증가
- 레거시 시스템: 오래된 시스템, 패치 어려움
- 가용성 우선: 보안보다 운영 연속성 중시
- 물리적 영향: 사이버 공격이 물리적 피해로 연결

## 목적
- 스마트팩토리 보안 위협 식별
- IT/OT 통합 보안 체계 구축
- 산업제어시스템 보안 강화

## 구성요소
| 구분 | 취약점 | 대응방안 |
|------|--------|----------|
| 네트워크 | IT/OT 망 연결 | 망분리, DMZ 구성 |
| 네트워크 | 평문 프로토콜 | 암호화 프로토콜 적용 |
| 시스템 | 레거시 취약점 | 가상 패치, 격리 |
| 시스템 | 디폴트 설정 | 보안 설정 강화 |
| 디바이스 | PLC 취약점 | 펌웨어 업데이트, 인증 |
| 디바이스 | 센서 조작 | 무결성 검증 |
| 인증 | 인증 미흡 | 강화된 인증, MFA |
| 모니터링 | 이상 탐지 부재 | OT 전용 IDS/IPS |
