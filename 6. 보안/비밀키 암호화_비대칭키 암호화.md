## 정의
키 사용 방식에 따른 암호화 분류, 비밀키/비대칭키 암호화
- 비밀키(대칭키) 암호화: 암호화와 복호화에 동일한 키를 사용하는 방식
- 비대칭키(공개키) 암호화: 암호화와 복호화에 서로 다른 키(공개키/개인키)를 사용하는 방식

## 키워드
* 대칭키, 공개키, 개인키, DES, AES, RSA, ECC, 키 분배, 전자서명

## 암기법
* 대스블: (대칭키) 스트림암호, 블록암호
* 이소타전: (비대칭키) 이산대수, 소인수분해, 타원곡선, 전자서명
* 속키부: (비교) 속도-대칭키 빠름, 키관리-비대칭키 용이, 부인방지-비대칭키

## 특징
| 구분 | 대칭키(비밀키) | 비대칭키(공개키) |
|------|----------------|------------------|
| 키 개수 | 1개 (동일 키) | 2개 (공개키/개인키) |
| 속도 | 빠름 | 느림 |
| 키 분배 | 어려움 | 용이 |
| 키 관리 | n(n-1)/2 | 2n |
| 부인방지 | 불가 | 가능 |
| 용도 | 대용량 데이터 암호화 | 키 교환, 전자서명 |

## 목적
- 대칭키: 대용량 데이터의 빠른 암호화
- 비대칭키: 안전한 키 교환 및 전자서명

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 대칭키 | 스트림 암호 | RC4, 비트 단위 암호화 |
| 대칭키 | 블록 암호 | DES, AES, SEED, ARIA |
| 비대칭키 | 소인수분해 기반 | RSA |
| 비대칭키 | 이산대수 기반 | Diffie-Hellman, ElGamal |
| 비대칭키 | 타원곡선 기반 | ECC (Elliptic Curve Cryptography) |
| 하이브리드 | 전자봉투 | 대칭키+비대칭키 결합 |

## 구성도
```
              [암호화 방식]
                   │
     ┌─────────────┴─────────────┐
     ▼                           ▼
[대칭키 암호화]            [비대칭키 암호화]
(동일 키 사용)             (공개키/개인키)
     │                           │
 ┌───┴───┐               ┌───────┼───────┐
 ▼       ▼               ▼       ▼       ▼
[스트림] [블록]        [RSA]   [ECC]  [ElGamal]
(RC4)  (AES,DES)    (소인수) (타원곡선)(이산대수)
     │                           │
     └───────────┬───────────────┘
                 ▼
          [하이브리드 암호화]
          (전자봉투: 대칭키로 데이터 암호화,
           공개키로 대칭키 암호화)
```
