#정보관리기술사 #보안 #포렌식 #아티팩트 #정리 #보안/개발/포렌식

## 정의
디지털 증거의 흔적, 디지털 포렌식 아티팩트(Artifact)
- 컴퓨터나 디지털 기기에서 사용자의 행위나 시스템 동작의 흔적을 보여주는 데이터나 파일 조각
- 시스템, 응용 프로그램, 파일 시스템에서 생성되는 다양한 형태의 디지털 증거물

## 키워드
* 레지스트리, 이벤트 로그, 프리패치, 브라우저 히스토리, 메타데이터, MFT

## 암기법
* 시응파: (아티팩트 유형) 시스템 아티팩트, 응용프로그램 아티팩트, 파일 아티팩트

## 특징
- 흔적 기반: 사용자/시스템 활동의 흔적
- 다양한 위치: 레지스트리, 로그, 캐시 등
- 시간 정보: 타임스탬프 포함
- 복구 가능: 삭제된 데이터도 복구 가능

## 목적
- 사용자 행위 추적 및 분석
- 악성코드 감염 경로 파악
- 범죄 증거 확보 및 타임라인 구성

## 구성요소
| 구분 | 아티팩트 | 설명 |
|------|----------|------|
| 시스템 | 레지스트리 | Windows 시스템/사용자 설정 정보 |
| 시스템 | 이벤트 로그 | 시스템/보안/응용프로그램 로그 |
| 시스템 | 프리패치 | 실행 파일 로딩 최적화 정보 |
| 시스템 | 점프 리스트 | 최근 사용 파일/프로그램 목록 |
| 응용프로그램 | 브라우저 히스토리 | 웹 방문 기록, 쿠키, 캐시 |
| 응용프로그램 | 이메일 | 메일 헤더, 첨부파일 |
| 응용프로그램 | 메신저 로그 | 대화 내역, 파일 전송 기록 |
| 파일 | MFT | Master File Table, 파일 메타데이터 |
| 파일 | 타임스탬프 | 생성/수정/접근 시간 |
| 파일 | 슬랙 공간 | 파일 할당 후 남은 공간 |

## 구성도
```
[디지털 포렌식 아티팩트]
         │
    ┌────┴────┬────────────┐
    ▼         ▼            ▼
[시스템     [응용프로그램  [파일

    │         │            │
    ▼         ▼            ▼
[레지스트리][브라우저]    [MFT]
[이벤트로그][이메일]      [타임스탬프]
[프리패치]  [메신저]      [슬랙공간]

    │
    ▼

    ├── 네트워크 접속 시점
    └── 사용자 로그인/로그아웃
```
