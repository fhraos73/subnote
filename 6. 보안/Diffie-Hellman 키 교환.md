#정보관리기술사 #보안 #암호화 #키교환 #옵션 #정리

## 정의
이산대수 기반 비밀키 공유 프로토콜, Diffie-Hellman 키 교환
- 암호화되지 않은 통신망을 통해 두 사람이 공통의 비밀키를 안전하게 공유하도록 하는 키 교환 알고리즘
- 이산대수 문제의 어려움에 기반한 최초의 공개키 기반 키 교환 방식

## 키워드
* 이산대수, 공개키, 비밀키, 키 교환, 중간자 공격, g, p, 모듈러 연산

## 암기법
* 이공비중: (특징) 이산대수, 공개키 사용, 비밀키 공유, 중간자 공격 취약

## 특징
- 이산대수 기반: g^x mod p 계산은 쉬우나 역산은 어려움
- 공개키 사용: 공개된 채널로 키 교환 가능
- 비밀키 생성: 양측이 동일한 비밀키 도출
- 중간자 공격 취약: 인증 메커니즘 부재 시 취약

## 목적
- 안전하지 않은 채널에서 비밀키 공유
- 대칭키 암호화를 위한 세션 키 생성
- SSL/TLS 등 보안 프로토콜의 키 교환

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 공개값 | p | 큰 소수 (1024bit 이상) |
| 공개값 | g | 연산에 사용되는 베이스(생성자) |
| 개인값 | x, y | 각 참여자의 비밀 정수 |
| 교환값 | R1 = g^x mod p | Alice가 Bob에게 전송 |
| 교환값 | R2 = g^y mod p | Bob이 Alice에게 전송 |
| 비밀키 | K = R2^x = R1^y mod p | 양측이 계산한 동일 비밀키 |

## 구성도
```
[사전 준비: g, p 공유]
         │
    ┌────┴────┐
    ▼         ▼
[Alice]     [Bob]
x 선택      y 선택
    │         │
    ▼         ▼
R1=g^x mod p  R2=g^y mod p
    │         │
    └────┬────┘
    (R1, R2 교환)
    ┌────┴────┐
    ▼         ▼
K=R2^x mod p  K=R1^y mod p
    │         │
    └────┬────┘
         ▼
   [동일 비밀키 K]
   K = g^(xy) mod p
```
