#정보관리기술사 #보안 #포렌식 #이미징 #정리

## 정의
저장매체 비트 단위 복제 기술, 디스크 이미징(Disk Imaging)
- 디지털 포렌식에서 원본 저장매체의 모든 데이터를 비트 단위로 완전히 복제하여 사본을 생성하는 기술
- 원본 증거의 무결성을 보존하면서 분석용 사본을 확보하는 핵심 기술

## 키워드
* 비트 복제, 해시값, 무결성, Write Blocker, E01, DD, 물리적 이미징

## 암기법
* 물논: (이미징 방식) 물리적 이미징, 논리적 이미징
* DD E01 AFF: (이미지 포맷) DD(Raw), E01(EnCase), AFF(Advanced Forensic Format)

## 특징
- 비트 단위 복제: 삭제된 데이터, 슬랙 공간 포함
- 무결성 보장: 해시값으로 원본과 사본 일치 검증
- 쓰기 방지: Write Blocker로 원본 변조 방지
- 법적 증거력: 원본과 동일한 증거 가치

## 목적
- 원본 증거의 무결성 보존
- 분석용 사본 확보
- 다수 분석자의 동시 분석 지원

## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 방식 | 물리적 이미징 | 전체 디스크 비트 단위 복제 |
| 방식 | 논리적 이미징 | 파일 시스템 단위 복제 |
| 방식 | Disk to Disk | 디스크 간 직접 복제 |
| 방식 | Disk to File | 이미지 파일로 저장 |
| 포맷 | DD (Raw) | 원시 비트스트림, 압축 없음 |
| 포맷 | E01 (EnCase) | 압축, 해시, 메타데이터 포함 |
| 포맷 | AFF | 오픈 포맷, 압축, 암호화 지원 |
| 도구 | Write Blocker | 원본 쓰기 방지 장치 |
| 검증 | MD5/SHA-256 | 해시값 비교로 무결성 검증 |

## 구성도
```
[디스크 이미징 절차]

[원본 디스크] ──▶ [Write Blocker] ──▶ [이미징 도구]
                  (쓰기 방지)           │
                                       ▼
                              [이미지 파일 생성]
                              (DD, E01, AFF)
                                       │
                              [해시값 생성]
                              (MD5, SHA-256)
                                       │
                              [해시값 비교]
                              (원본 = 사본)
                                       │
                              [무결성 검증 완료]

[이미징 방식 비교]
┌─────────────┬─────────────┐
│ 물리적 이미징 │ 논리적 이미징 │
├─────────────┼─────────────┤
│ 전체 디스크   │ 파일 시스템   │
│ 슬랙 포함    │ 파일만 복제   │
│ 삭제파일 포함 │ 빠른 속도    │
│ 시간 소요    │ 증거 누락 가능│
└─────────────┴─────────────┘
```
