#정보관리기술사 #보안 #클라우드 #CSAP #인증 #정리

## 정의
클라우드 서비스 보안 인증제도, CSAP
- 클라우드 서비스 제공자가 제공하는 서비스에 대해 정보보호 기준의 준수 여부를 평가·인증하는 제도
- 공공기관의 클라우드 서비스 도입 시 보안성 검증을 위한 국내 인증 제도

## 키워드
* 클라우드 보안, 공공기관, KISA, 보안인증, IaaS, SaaS, PaaS, 보안등급

## 암기법
* 관물기가운침: (평가항목) 관리적 보안, 물리적 보안, 기술적 보안, 가상화 보안, 운영 보안, 침해사고 관리

## 특징
- 공공기관 필수: 공공기관 클라우드 도입 시 필수 인증
- 서비스 유형별: IaaS, SaaS, PaaS 별 평가 기준
- 등급제: 상/중/하 보안 등급 분류
- 연간 사후관리: 인증 유지를 위한 정기 점검

## 목적
- 클라우드 서비스의 보안성 검증
- 공공기관의 안전한 클라우드 도입 지원
- 클라우드 서비스 제공자의 보안 수준 향상

## 구성요소
| 구분     | 평가항목    | 설명               |
| ------ | ------- | ---------------- |
| 관리적 보안 | 정보보호 정책 | 보안 정책, 조직, 자산 관리 |
| 관리적 보안 | 인적 보안   | 보안 교육, 인력 관리     |
| 물리적 보안 | 시설 보안   | 데이터센터 물리적 보안     |
| 기술적 보안 | 접근 통제   | 인증, 권한 관리        |
| 기술적 보안 | 암호화     | 데이터 암호화, 키 관리    |
| 가상화 보안 | 하이퍼바이저  | 가상화 환경 보안        |
| 가상화 보안 | 가상 네트워크 | 가상 네트워크 분리       |
| 운영 보안  | 취약점 관리  | 보안 패치, 취약점 점검    |
| 침해사고   | 대응 체계   | 침해사고 탐지, 대응, 복구  |
