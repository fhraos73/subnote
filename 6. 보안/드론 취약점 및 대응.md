#정보관리기술사 #보안 #드론 #UAV #취약점 #정리 #보안/IoT/신기술

## 정의
무선 전파 유도 비행 무인항공기, 드론(UAV)
- 무선 전파의 유도에 의해서 비행하는 무인항공기(UAV, Unmanned Aerial Vehicle)의 총칭
- GPS, RF 통신, 센서 기반 자율/원격 비행 시스템

## 키워드
* GPS Spoofing, GPS Jamming, 도청공격, 악성코드 감염, 통신 암호화, 안티드론

## 암기법
* 통방도악 GPS불불: (위협요소) 통신신호속이기, 통신방해, 통신도청, 악성코드감염 / GPS Spoofing, GPS Jamming, 불법탈취후키해킹, 불법장치탑재
* 관기물: (대응방안) 관리적-로그/인증/관제/보안진단/교육, 기술적-상호인증/암호화/접근통제, 물리적-탈취대응/장치제한/위협완화

## 특징
- 무선 통신 의존: RF 통신 취약점 노출
- GPS 의존: GPS 스푸핑/재밍 취약
- 원격 제어: 제어 신호 가로채기 위험
- 물리적 위협: 충돌, 추락 등 물리적 피해

| 구분 | 문제점 | 설명 |
|------|--------|------|
| 정보보호 | 개인정보 침해 | 카메라를 통한 사생활 침해 |
| 정보보호 | 범죄 악용 | 불법 촬영, 밀수 등 범죄 수단 |
| 군사적 | 정보 탈취 | 국가 주요시설 정보 탈취 |

| 추락 | 재산/인명 피해 | 추락으로 인한 물리적 피해 |

| 통신 | 통신신호 속이기 | GPS Spoofing | 탈취 |

| 통신 | 통신도청 | 도청공격 | 정보유출 |

| 물리 | 불법 탈취 | 물리적 탈취 후 키 해킹 | 제어권 탈취 |
| 물리 | 불법 장치 탑재 | 폭발물, 도청장치 부착 | 테러, 정보유출 |

|------|----------|------|
| 관리적 | 로그 의무화 | 비행 기록 의무 저장 및 관리 |
| 관리적 | 인증 강화 | 드론 및 조종자 인증 체계 |
| 관리적 | 드론관제센터 구축 | 중앙 집중식 드론 관제 시스템 |

| 관리적 | 교육/자격증 | 조종자 보안 교육 및 자격 관리 |
| 기술적 | GPS 상호인증 | GPS 송/수신기 간 상호 인증 |
| 기술적 | 통신 암호화 | 제어 명령 및 데이터 암호화 |
| 기술적 | 접근통제 | 인가된 사용자만 제어 허용 |
| 물리적 | 탈취 대응 | 물리적 장비 탈취 방지 장치 |
| 물리적 | 장치 탑재 제한 | 불법 장치 탑재 탐지/방지 |
| 물리적 | 위협 완화 | 지오펜싱, 자동 귀환 등 |
