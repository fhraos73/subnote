#디지털네트워크 #QoS #WFQ #필수 #디지털네트워크/QoS
## 정의
가중치 공정 큐잉(WFQ)
- IP Precedence 기반 가중치를 적용하여 플로우별 공정한 대역폭 분배를 보장하는 큐잉 알고리즘
- GPS(Generalized Processor Sharing)를 패킷 레벨에서 구현한 스케줄링 기법
## 키워드
* GPS, PGPS, Virtual Finish Time
* IP Precedence, Weight, Flow
* O(log n) 복잡도, Starvation Free
## 암기법
* "가공플" - 가중치, 공정, 플로우
* "아이피 가중 공정" - IP Precedence 기반 가중치 공정 분배
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| 가중치 할당 | w_i / Σw × R (링크 대역폭 비례 분배) | IP Precedence 활용 |
| Virtual Finish Time | 패킷별 가상 완료시간 계산 | 최소값 우선 전송 |
| Flow 분류 | 자동 플로우 식별 및 큐 생성 | 최대 256개 큐 |
| 기아방지 | 모든 플로우 최소 대역폭 보장 | Starvation Free |
| 지터 감소 | 플로우별 독립 큐로 지터 최소화 | 실시간 적합 |
| 공격적 플로우 제어 | 과다 트래픽 플로우 우선 Drop | Tail Drop 개선 |
