#디지털네트워크 #QoS #큐관리 #필수 #디지털네트워크/QoS
## 정의
네트워크 QoS 큐 관리 기법
- 네트워크 트래픽의 서비스 품질 보장을 위한 패킷 버퍼링 및 전송 순서 결정 메커니즘
- 대역폭 할당, 지연 최소화, 패킷 손실 방지를 위한 큐잉 스케줄링 기술
## 키워드
* FIFO, PQ, WFQ, CQ, CBWFQ, LLQ
* Tail Drop, Head of Line Blocking
* Scheduling, Queuing Discipline
## 암기법
* "피프큐씨" - FIFO, PQ, WFQ, CQ (기본 4대 큐잉)
* "선공가" - 선입선출, 공정큐잉, 가중치큐잉
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| FIFO | 선입선출, 도착순 처리 | 단순, Tail Drop |
| PQ | 우선순위별 별도 큐, 높은 우선순위 먼저 처리 | 기아현상 발생 가능 |
| WFQ | IP Precedence 기반 가중치 대역폭 할당 | 공정성 보장 |
| CQ | 16개 분류 큐, Round-Robin 처리 | 지연 발생 |
| CBWFQ | 클래스별 대역폭 보장 | WFQ 확장 |
| LLQ | PQ + CBWFQ 결합, 실시간 트래픽 우선 | VoIP 적합 |
