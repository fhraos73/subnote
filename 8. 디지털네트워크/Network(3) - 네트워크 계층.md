#디지털네트워크 #네트워크계층 #OSI #옵션 #디지털네트워크/OSI/프로토콜
## 정의
논리적 주소 기반 경로 설정 계층, Network Layer
- OSI 7계층 중 3계층으로 패킷의 경로 설정(Routing) 담당
- IP 주소를 사용하여 네트워크 간 데이터 전달 수행
## 키워드
* IP, 라우팅, 패킷, 논리주소, ICMP, ARP, 라우터, 포워딩
## 암기법
* 네트워크 계층 기능: 라포주분(라우팅, 포워딩, 주소지정, 분할/재조립)
* 라우팅 프로토콜: 리오아비(RIP, OSPF, IGRP, BGP)
## 구성요소
| 구분 | 설명 | 비고 |
|------|------|------|
| IP | 논리적 주소 체계 | IPv4, IPv6 |
| ICMP | 오류 보고 및 진단 | ping, traceroute |
| ARP | IP→MAC 주소 변환 | Address Resolution |
| RARP | MAC→IP 주소 변환 | Reverse ARP |
| 라우터 | 경로 설정 장비 | 패킷 포워딩 |
| 패킷 | 네트워크 계층 PDU | IP 헤더+데이터 |
## 주요 기능
| 기능 | 설명 | 비고 |
|------|------|------|
| 라우팅 | 최적 경로 결정 | 라우팅 테이블 |
| 포워딩 | 패킷 전달 | 다음 홉 결정 |
| 주소지정 | 논리적 주소 할당 | IP 주소 |
| 분할/재조립 | MTU 기반 분할 | Fragmentation |
| 혼잡제어 | 네트워크 혼잡 관리 | QoS |
## 구성도
```
[네트워크 계층 위치]
┌─────────────────────────────────────┐
│     Transport Layer (4)             │
├─────────────────────────────────────┤
│     Network Layer (3)               │
│  ┌────────────────────────────────┐ │
│  │  IP │ ICMP │ ARP │ RARP │ IGMP│ │
│  └────────────────────────────────┘ │
├─────────────────────────────────────┤
│     Data Link Layer (2)             │
└─────────────────────────────────────┘

[라우팅 vs 포워딩]
┌─────────┐    라우팅 테이블    ┌─────────┐
│ Network │◀───────────────────│ Router  │
│    A    │     (경로 결정)     │         │
└─────────┘                    └────┬────┘
                                   │포워딩
                               ┌───▼────┐
                               │Network │
                               │   B    │
                               └────────┘

[라우팅 프로토콜 분류]
IGP (내부)              EGP (외부)
├─ Distance Vector      └─ BGP
│  └─ RIP, IGRP
└─ Link State
   └─ OSPF, IS-IS
```
