#디지털네트워크 #QoS #DiffServ #필수 #디지털네트워크/QoS
## 정의
차등 서비스(DiffServ)
- DSCP 기반 클래스별 차등 서비스를 제공하는 Coarse-grained QoS 아키텍처
- Edge에서 분류/마킹, Core에서 PHB 적용으로 확장성을 확보한 RFC 2474 기반 모델
## 키워드
* DSCP, DS Field, PHB
* EF, AF, BE, Class Selector
* Edge Router, Core Router
## 암기법
* "디디피" - DiffServ, DSCP, PHB
* "에에베" - EF(긴급), AF(보장), BE(최선)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| DSCP | 6bit 코드포인트, 64개 클래스 | IP 헤더 DS Field |
| PHB | Per-Hop Behavior, 홉별 처리 동작 | 라우터 정책 |
| EF | Expedited Forwarding, 최우선 처리 | 저지연/저손실 |
| AF | Assured Forwarding, 4클래스×3드롭 | 12개 조합 |
| BE | Best Effort, 기본 서비스 | DSCP=0 |
| Edge Router | 트래픽 분류, DSCP 마킹 | 경계 라우터 |
| Core Router | PHB만 적용, 분류 없음 | 고속 처리 |

│ ──┼─>│ Router │──>│ Router │──>│ Router │──>       │  │

│                DSCP/PHB 매핑                           │
