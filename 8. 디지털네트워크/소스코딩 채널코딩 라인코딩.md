#디지털네트워크 #통신이론 #코딩 #옵션
## 정의
디지털 통신의 3대 코딩, 소스/채널/라인 코딩
- 소스코딩: 데이터 압축으로 전송 효율 향상
- 채널코딩: 오류 정정 코드 추가로 신뢰성 확보
- 라인코딩: 전송 매체에 적합한 신호 변환
## 키워드
* 소스코딩(압축), 채널코딩(FEC), 라인코딩(신호변환), 샤논 정리, 엔트로피, 채널 용량
## 암기법
* "소압채오라신" - 소스(압축), 채널(오류정정), 라인(신호변환)
## 구성요소/특징/유형
### 소스코딩 (Source Coding)
| 구분 | 설명 | 예시 |
|------|------|------|
| 목적 | 데이터 압축, 중복 제거 | 전송량 감소 |
| 무손실 | 원본 복원 가능 | Huffman, LZW, ZIP |
| 손실 | 일부 정보 손실 | JPEG, MP3, H.264 |
| 이론 | 샤논 엔트로피 | H(X) = -Σp(x)log₂p(x) |
### 채널코딩 (Channel Coding)
| 구분 | 설명 | 예시 |
|------|------|------|
| 목적 | 오류 검출/정정 | 신뢰성 향상 |
| 블록 코드 | 고정 길이 코드워드 | Hamming, Reed-Solomon |
| 컨볼루션 | 연속 인코딩 | Viterbi 디코딩 |
| 터보/LDPC | 고성능 FEC | 4G/5G, Wi-Fi |
| 이론 | 샤논 채널 용량 | C = B·log₂(1+SNR) |
### 라인코딩 (Line Coding)
| 구분 | 설명 | 특징 |
|------|------|------|
| NRZ | Non-Return to Zero | 단순, DC 성분 |
| RZ | Return to Zero | 클럭 복원 용이 |
| Manchester | 비트 중간 천이 | 자기 동기화 |
| 4B/5B | 4비트→5비트 변환 | DC 균형 |
| 8B/10B | 8비트→10비트 변환 | 기가비트 이더넷 |
## 구성도
```
┌─────────────────────────────────────────────────────────────┐
│                디지털 통신 시스템 구조                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [송신측]                                                    │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐ │
│  │  원본    │ → │ 소스코딩  │ → │ 채널코딩  │ → │ 라인코딩  │ │
│  │  데이터  │   │  (압축)   │   │(오류정정) │   │(신호변환) │ │
│  └──────────┘   └──────────┘   └──────────┘   └────┬─────┘ │
│                                                     │       │
│                                                     ↓       │
│                                              ┌──────────┐   │
│                                              │   채널   │   │
│                                              │  (전송)  │   │
│                                              └────┬─────┘   │
│                                                   │         │
│  [수신측]                                          ↓         │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐ │
│  │  복원    │ ← │ 소스디코딩│ ← │ 채널디코딩│ ← │ 라인디코딩│ │
│  │  데이터  │   │  (압축해제)│   │(오류정정) │   │(신호복원) │ │
│  └──────────┘   └──────────┘   └──────────┘   └──────────┘ │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                    샤논 정리 (Shannon's Theorem)             │
│                                                             │
│  [샤논 엔트로피] - 소스코딩 한계                              │
│  H(X) = -Σ p(x) · log₂p(x)  (bits/symbol)                  │
│  → 무손실 압축의 이론적 한계                                 │
│                                                             │
│  [샤논 채널 용량] - 채널코딩 한계                             │
│  C = B · log₂(1 + S/N)  (bits/second)                      │
│  C: 채널 용량, B: 대역폭, S/N: 신호 대 잡음비                │
│  → 오류 없는 전송의 이론적 최대 속도                         │
│                                                             │
│  [라인코딩 비교]                                             │
│  NRZ-L:    ─┐  ┌──┐     ┌──                                │
│            └──┘  └─────┘                                    │
│  Manchester: ┌┐┌┐  ┌┐    ┌┐┌┐                              │
│              └┘└┘  └┘    └┘└┘                              │
│             (비트 중간에서 천이 발생)                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
