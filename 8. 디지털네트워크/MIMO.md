#디지털네트워크 #안테나 #무선통신 #필수
## 정의
다중입력 다중출력, MIMO
- 송수신 양측에 다중 안테나를 사용하여 동시에 여러 신호를 전송하는 기술
- 다중경로 페이딩을 활용하여 주파수/전력 증가 없이 용량 증대
## 키워드
* 공간 다중화(Spatial Multiplexing), 공간 다이버시티, 빔포밍
* Massive MIMO, MU-MIMO, 채널 용량, 안테나 배열
## 암기법
* "공공빔" - 공간다중화, 공간다이버시티, 빔포밍
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| 공간 다중화 | 각 안테나가 독립 데이터 전송 | 용량 증대 |
| 공간 다이버시티 | 동일 데이터 다중 경로 전송 | 신뢰성 향상 |
| 빔포밍 | 특정 방향으로 신호 집중 | 간섭 감소 |
| SU-MIMO | 단일 사용자 MIMO | 4x4, 8x8 |
| MU-MIMO | 다중 사용자 MIMO | 동시 다중 사용자 |
| Massive MIMO | 수십~수백 안테나 | 5G 핵심 기술 |
## 구성도
```
[송신측]                              [수신측]
┌──────────────┐                     ┌──────────────┐
│   TX 1 ○────┼──┐              ┌──┼────○ RX 1   │
│              │  │    다중경로   │  │              │
│   TX 2 ○────┼──┼──────────────┼──┼────○ RX 2   │
│              │  │   (페이딩)   │  │              │
│   TX 3 ○────┼──┼──────────────┼──┼────○ RX 3   │
│              │  │              │  │              │
│   TX 4 ○────┼──┘              └──┼────○ RX 4   │
└──────────────┘                     └──────────────┘
     N개 TX                              M개 RX
     
     용량 = min(N,M) × SISO 용량
```
