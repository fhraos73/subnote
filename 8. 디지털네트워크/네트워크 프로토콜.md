#디지털네트워크 #프로토콜 #통신규약 #옵션 #디지털네트워크/OSI/프로토콜
## 정의
네트워크 통신 표준 규약, Network Protocol
- 컴퓨터 네트워크에서 데이터를 교환하기 위해 정의된 통신 규칙과 절차의 집합
- 구문(Syntax), 의미(Semantics), 타이밍(Timing)의 3요소로 구성
## 키워드
* 구문, 의미, 타이밍, TCP/IP, OSI, 캡슐화, PDU, 프로토콜 스택
## 암기법
* 프로토콜 3요소: 구의타(구문, 의미, 타이밍)
* TCP/IP 4계층: 네인전응(네트워크접근, 인터넷, 전송, 응용)
## 프로토콜 3요소
| 요소 | 설명 | 예시 |
|------|------|------|
| 구문(Syntax) | 데이터 형식, 부호화, 신호 레벨 | 헤더 구조, 필드 크기 |
| 의미(Semantics) | 제어 정보, 오류 처리 방법 | 플래그 의미, 오류 코드 |
| 타이밍(Timing) | 속도 조절, 순서 제어 | 전송 속도, 응답 시간 |
## 주요 프로토콜
| 계층 | 프로토콜 | 설명 |
|------|----------|------|
| 응용 | HTTP, FTP, SMTP | 웹, 파일전송, 메일 |
| 응용 | DNS, DHCP | 도메인, IP할당 |
| 전송 | TCP, UDP | 신뢰성/비신뢰성 전송 |
| 네트워크 | IP, ICMP, ARP | 주소지정, 오류보고 |
| 데이터링크 | Ethernet, PPP | LAN, WAN 프로토콜 |
