#디지털네트워크 #5G #특화망 #이음5G #필수 #디지털네트워크/5G/6G
## 정의
5G 특화망 (Private 5G)
- 특정 기업/기관 전용으로 구축하는 독립적인 5G 네트워크 인프라
- 이음5G로 명명된 국내 정책으로 제조, 물류, 의료 등 산업 DX를 지원하는 전용망
## 키워드
* Private 5G, Local 5G, 이음5G
* 4.7GHz, 28GHz 전용 주파수
* MEC, On-Premise, 보안
## 암기법
* "특전보" - 특정용도, 전용주파수, 보안강화
* "이음오지" - 이음5G (끊김없는 연결)
## 구성요소/특징/유형
| 구분         | 설명                            | 비고        |
| ---------- | ----------------------------- | --------- |
| 독립형(SA)    | 자체 코어망 구축, 완전 독립              | 높은 보안     |
| 연동형(NSA)   | 통신사 코어 활용, 무선만 전용             | 빠른 구축     |
| 전용주파수      | 4.7GHz(100MHz), 28GHz(800MHz) | 간섭 없음     |
| On-Premise | 사업장 내 장비 설치                   | 데이터 내부 유지 |
| MEC 연동     | 엣지 컴퓨팅으로 저지연 처리               | 실시간 제어    |
| PQC 보안     | 양자내성암호 적용                     | 미래 보안 대비  |
