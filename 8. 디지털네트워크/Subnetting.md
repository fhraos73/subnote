#디지털네트워크 #서브넷 #IP주소 #옵션 #디지털네트워크/TCP/UDP
## 정의
IP 네트워크 분할 기법, Subnetting
- 하나의 네트워크를 여러 개의 작은 서브네트워크로 분할하는 기법
- 서브넷 마스크를 이용하여 네트워크 ID와 호스트 ID 경계를 조정
## 키워드
* 서브넷마스크, CIDR, 네트워크ID, 호스트ID, 브로드캐스트, 게이트웨이, VLSM
## 암기법
* 서브넷 계산: 네호브게(네트워크주소, 호스트범위, 브로드캐스트, 게이트웨이)
* 클래스: A(8), B(16), C(24) - 기본 서브넷 마스크 비트 수
## IP 주소 클래스
| 클래스 | 범위 | 기본 마스크 | 네트워크/호스트 |
|--------|------|-------------|-----------------|
| A | 1~126 | 255.0.0.0 (/8) | 126개/16,777,214개 |
| B | 128~191 | 255.255.0.0 (/16) | 16,384개/65,534개 |
| C | 192~223 | 255.255.255.0 (/24) | 2,097,152개/254개 |
| D | 224~239 | - | 멀티캐스트 |
| E | 240~255 | - | 예약 |
## 서브넷 계산
| 항목 | 공식 | 예시 (/26) |
|------|------|------------|
| 서브넷 수 | 2^n (n=빌린비트) | 2^2 = 4개 |
| 호스트 수 | 2^h - 2 (h=호스트비트) | 2^6 - 2 = 62개 |
| 블록 크기 | 256 - 서브넷마스크 | 256 - 192 = 64 |
| 네트워크 주소 | 블록 시작 | .0, .64, .128, .192 |
| 브로드캐스트 | 다음 블록 - 1 | .63, .127, .191, .255 |
## 구성도
```
[IP 주소 구조]
┌─────────────────────────────────────────────────┐
│              32 bits IP Address                 │
├─────────────────────┬───────────────────────────┤
│    Network ID       │       Host ID             │
└─────────────────────┴───────────────────────────┘

[서브넷팅 예시: 192.168.1.0/24 → /26]
원본: 192.168.1.0/24 (호스트 254개)
      ┌─────────────────────────────────────┐
      │         192.168.1.0 ~ .255          │
      └─────────────────────────────────────┘

서브넷팅 후: /26 (4개 서브넷, 각 62개 호스트)
┌─────────────────┬─────────────────┐
│ 192.168.1.0/26  │ 192.168.1.64/26 │
│ .1 ~ .62        │ .65 ~ .126      │
│ BC: .63         │ BC: .127        │
├─────────────────┼─────────────────┤
│192.168.1.128/26 │192.168.1.192/26 │
│ .129 ~ .190     │ .193 ~ .254     │
│ BC: .191        │ BC: .255        │
└─────────────────┴─────────────────┘

[서브넷 마스크 표기]
CIDR    서브넷 마스크        호스트 수
/24     255.255.255.0        254
/25     255.255.255.128      126
/26     255.255.255.192      62
/27     255.255.255.224      30
/28     255.255.255.240      14
/29     255.255.255.248      6
/30     255.255.255.252      2

[VLSM (Variable Length Subnet Mask)]
┌─────────────────────────────────────────────────┐
│              192.168.1.0/24                     │
├─────────────────────────────────────────────────┤
│ /26 (62호스트) │ /27 (30) │ /28 (14) │ /30 (2) │
│   본사         │  지사1   │  지사2   │ P2P링크 │
└─────────────────────────────────────────────────┘
```
