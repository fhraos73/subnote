#디지털네트워크 #QoS #혼잡제어 #필수
## 정의
네트워크 혼잡제어
- 네트워크 혼잡 발생 시 송신률을 조절하여 패킷 손실과 지연을 최소화하는 메커니즘
- TCP 기반 End-to-End 제어와 라우터 기반 AQM(Active Queue Management) 기법 포함
## 키워드
* AIMD, Slow Start, Congestion Avoidance
* CWND, ssthresh, RTT
* RED, ECN, Tail Drop
## 암기법
* "슬혼빠빠" - Slow Start, 혼잡회피, 빠른재전송, 빠른회복
* "에이아이엠디" - Additive Increase, Multiplicative Decrease
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| Slow Start | CWND 지수 증가 (1→2→4→8) | ssthresh까지 |
| Congestion Avoidance | CWND 선형 증가 (+1/RTT) | 혼잡 감지 전 |
| Fast Retransmit | 3 Dup ACK 시 즉시 재전송 | 타임아웃 대기 없음 |
| Fast Recovery | 혼잡 시 CWND 절반으로 감소 | Slow Start 회피 |
| RED | 큐 임계치 기반 확률적 Drop | 조기 혼잡 감지 |
| ECN | IP 헤더 마킹으로 혼잡 알림 | Drop 없이 통보 |
## 구성도
```
┌─────────────────────────────────────────────────────────┐
│                TCP 혼잡제어 상태 전이                    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌────────────┐    ssthresh 도달    ┌──────────────┐  │
│   │ Slow Start │ ─────────────────> │ Congestion   │  │
│   │ (지수증가) │                     │ Avoidance    │  │
│   └────────────┘                     │ (선형증가)   │  │
│         │                            └──────────────┘  │
│         │ Timeout                           │          │
│         ▼                            3 Dup ACK        │
│   ┌────────────┐                           │          │
│   │  ssthresh  │ <─────────────────────────┘          │
│   │  = CWND/2  │                                      │
│   │  CWND = 1  │ ──> Fast Recovery (CWND=ssthresh)   │
│   └────────────┘                                      │
├─────────────────────────────────────────────────────────┤
│                AIMD 동작 그래프                         │
│   CWND                                                 │
│    ▲      /\      /\      /\                          │
│    │     /  \    /  \    /  \                         │
│    │    /    \  /    \  /    \                        │
│    │   /      \/      \/      \                       │
│    └──────────────────────────────> Time              │
│       선형증가    절반감소(혼잡)                        │
├─────────────────────────────────────────────────────────┤
│                AQM 기법 비교                           │
│  ┌──────────┬────────────────────────────────┐        │
│  │ Tail Drop│ 큐 Full 시 Drop, Global Sync   │        │
│  │ RED      │ 확률적 조기 Drop, 혼잡 예방    │        │
│  │ ECN      │ 패킷 마킹, Drop 없이 혼잡 통보 │        │
│  └──────────┴────────────────────────────────┘        │
└─────────────────────────────────────────────────────────┘
```
