#디지털네트워크 #네트워크가상화 #소프트웨어정의 #필수 #디지털네트워크/SDN/NFV
## 정의
소프트웨어 정의 네트워크, SDN
- 제어 평면과 데이터 평면을 분리하여 네트워크를 중앙집중식으로 제어하는 기술
- 프로그래머블 네트워크로 유연한 정책 적용 및 자동화 구현
## 키워드
* Control Plane/Data Plane 분리, SDN Controller, OpenFlow
* NBI(Northbound Interface), SBI(Southbound Interface), 프로그래머블
## 암기법
* "제데분 컨오노" - 제어/데이터 분리, 컨트롤러, 오픈플로우, 노스바운드
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| 애플리케이션 계층 | 네트워크 서비스/앱 | 모니터링, 보안, QoS |
| 컨트롤러 계층 | 중앙집중 제어 장치 | 네트워크의 두뇌 |
| 인프라 계층 | 데이터 전달 장비 | 스위치, 라우터 |
| NBI | 앱↔컨트롤러 인터페이스 | REST API |
| SBI | 컨트롤러↔인프라 인터페이스 | OpenFlow |
| 컨트롤러 | ODL, ONOS, Ryu | 오픈소스 |
