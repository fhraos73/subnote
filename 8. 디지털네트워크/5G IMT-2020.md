#디지털네트워크 #5G #IMT2020 #필수
## 정의
5G (IMT-2020)
- ITU-R이 정의한 5세대 이동통신 국제표준으로 eMBB, URLLC, mMTC 3대 서비스 시나리오 지원
- 최대 20Gbps, 1ms 지연, 100만 디바이스/km² 연결을 목표로 하는 차세대 무선 네트워크
## 키워드
* eMBB, URLLC, mMTC
* Massive MIMO, Beamforming, mmWave
* Network Slicing, MEC, NFV/SDN
## 암기법
* "이얼엠" - eMBB, URLLC, mMTC (3대 시나리오)
* "초저대" - 초고속, 저지연, 대규모 연결
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| eMBB | Enhanced Mobile Broadband, 초고속 | 20Gbps Peak |
| URLLC | Ultra-Reliable Low Latency | 1ms, 99.999% |
| mMTC | Massive Machine Type Comm. | 100만/km² |
| Massive MIMO | 대규모 안테나 어레이 | 64T64R 이상 |
| Beamforming | 빔 집중으로 커버리지 확대 | 간섭 감소 |
| mmWave | 밀리미터파 (24~100GHz) | 광대역 확보 |
| Network Slicing | 가상 네트워크 분리 | 서비스별 최적화 |
| MEC | Mobile Edge Computing | 저지연 처리 |
