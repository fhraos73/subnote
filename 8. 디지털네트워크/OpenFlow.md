#디지털네트워크 #SDN #프로토콜 #필수 #디지털네트워크/SDN/NFV
## 정의
SDN 표준 프로토콜, OpenFlow
- SDN 컨트롤러와 네트워크 장비 간 통신을 위한 표준 프로토콜
- ONF에서 관리하며 플로우 테이블 기반 패킷 포워딩 제어
## 키워드
* Flow Table, Match/Action/Counter, Packet-In/Out, Flow-Mod
* Proactive/Reactive, Pipeline, Group Table, ONF
## 암기법
* "플매액카 패플" - 플로우테이블, 매치, 액션, 카운터, 패킷인, 플로우모드
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| Flow Table | 패킷 매칭/포워딩 규칙 | Match+Action+Counter |
| Match | 패킷 헤더 필드 매칭 | MAC, IP, Port 등 |
| Action | 매칭 시 수행 동작 | Forward, Drop, Modify |
| Counter | 통계 정보 수집 | 패킷/바이트 수 |
| Packet-In | 미매칭 패킷→컨트롤러 | 질의 메시지 |
| Flow-Mod | 컨트롤러→플로우 설치 | 규칙 추가/수정 |
| Packet-Out | 컨트롤러→패킷 전송 | 직접 포워딩 |
