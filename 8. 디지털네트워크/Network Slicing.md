#디지털네트워크 #5G #NetworkSlicing #필수
## 정의
네트워크 슬라이싱
- 단일 물리 네트워크 인프라 위에 다수의 논리적 가상 네트워크를 생성하는 5G 핵심 기술
- SDN/NFV 기반으로 서비스별 맞춤형 E2E 네트워크를 동적으로 구성하는 기술
## 키워드
* SDN, NFV, VNF, CNF
* NSMF, NSSMF, CSMF
* RAN Slice, Core Slice, Transport Slice
## 암기법
* "슬에스" - Slicing, E2E, SDN/NFV
* "씨엔엔" - CSMF, NSMF, NSSMF (관리 계층)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| CSMF | Customer Service Management | 고객 요구→슬라이스 매핑 |
| NSMF | Network Slice Management | E2E 슬라이스 오케스트레이션 |
| NSSMF | Network Slice Subnet Mgmt | 도메인별 슬라이스 관리 |
| RAN Slice | 무선접속망 슬라이스 | gNB 자원 분할 |
| Core Slice | 코어망 슬라이스 | UPF/AMF 인스턴스 |
| Transport Slice | 전송망 슬라이스 | 백홀/프론트홀 |
