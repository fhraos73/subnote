#디지털네트워크 #네트워크가상화 #ETSI #필수
## 정의
네트워크 기능 가상화, NFV
- 전용 하드웨어 기반 네트워크 기능을 범용 서버에서 소프트웨어로 구현하는 기술
- ETSI NFV 아키텍처 기반으로 VNF, NFVI, MANO로 구성
## 키워드
* VNF(가상 네트워크 기능), NFVI(NFV 인프라), MANO(관리/오케스트레이션)
* VNFM, VIM, NFVO, 서비스 체이닝, ETSI
## 암기법
* "브이엔마" - VNF, NFVI, MANO
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| VNF | 가상화된 네트워크 기능 | 방화벽, 라우터, IDS |
| NFVI | NFV 인프라 | 서버, 스토리지, 네트워크 |
| MANO | 관리 및 오케스트레이션 | NFVO, VNFM, VIM |
| NFVO | NFV 오케스트레이터 | 서비스 생명주기 관리 |
| VNFM | VNF 매니저 | VNF 인스턴스 관리 |
| VIM | 가상 인프라 매니저 | NFVI 리소스 관리 |
