#디지털네트워크 #IoT #스마트홈 #필수 #디지털네트워크/Wi-Fi/IoT
## 정의
스마트홈 통합 표준, Matter
- CSA(Connectivity Standards Alliance)가 개발한 스마트홈 기기 간 상호운용성 표준 프로토콜
- Apple, Google, Amazon, Samsung 등이 참여하여 IoT 기기의 호환성 문제 해결
## 키워드
* CSA, Thread, Wi-Fi, Bluetooth, IP 기반, 상호운용성, 로컬 제어, 보안
## 암기법
* "씨쓰와블아" - CSA, Thread, Wi-Fi, Bluetooth, IP(아이피)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| IP 기반 | IPv6 기반 통신 | 인터넷 표준 활용 |
| Thread | 저전력 메시 네트워크 | IEEE 802.15.4 |
| Wi-Fi | 고대역폭 연결 | 기존 인프라 활용 |
| Bluetooth LE | 기기 설정/페어링 | 초기 설정용 |
| 로컬 제어 | 클라우드 없이 로컬 동작 | 프라이버시 강화 |
### Matter 지원 기기 유형
| 카테고리 | 기기 예시 |
|----------|-----------|
| 조명 | 전구, 스위치, 디머 |
| HVAC | 온도조절기, 에어컨 |
| 보안 | 도어락, 센서, 카메라 |
| 가전 | 로봇청소기, 냉장고 |
| 미디어 | TV, 스피커 |
