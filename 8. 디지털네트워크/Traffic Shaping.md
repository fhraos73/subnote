#디지털네트워크 #QoS #트래픽쉐이핑 #필수 #디지털네트워크/QoS
## 정의
트래픽 정형화(Shaping)
- 초과 트래픽을 버퍼에 저장 후 일정한 속도로 전송하여 출력을 평활화하는 트래픽 제어 기법
- 아웃바운드 트래픽에 적용되며 Leaky Bucket 알고리즘 기반 동작
## 키워드
* Leaky Bucket, Token Bucket
* Buffering, Smoothing, Delay
* Outbound, Tc(Time Interval)
## 암기법
* "쉐버스" - Shaping은 버퍼링 후 Smooth
* "초과저장평활" - 초과 트래픽 저장 후 평활화
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| Leaky Bucket | 일정 속도로 패킷 방출 | 출력 평활화 |
| Token Bucket | 토큰 기반 전송 제어 | 버스트 허용 |
| CIR | 평균 전송률 설정 | 목표 대역폭 |
| Tc | Time Interval, 전송 주기 | Bc/CIR |
| Bc | Committed Burst, 주기당 전송량 | CIR × Tc |
| Queue Depth | 버퍼 크기, 지연 결정 | 오버플로우 시 Drop |
