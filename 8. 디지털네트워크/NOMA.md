#디지털네트워크 #다중접속 #6G #필수 #디지털네트워크/무선기술
## 정의
비직교 다중접속, NOMA
- 동일 주파수 자원을 다중 사용자가 동시 사용하며 전력/코드 차이로 구분하는 기술
- SIC(연속간섭제거)를 통해 중첩된 신호를 순차적으로 분리
## 키워드
* 비직교(Non-Orthogonal), SIC(Successive Interference Cancellation)
* 전력 도메인, 코드 도메인, 스펙트럼 효율, 대규모 연결
## 암기법
* "비전코시" - 비직교, 전력도메인, 코드도메인, SIC
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| 전력 도메인 | 전력 차이로 사용자 구분 | PD-NOMA |
| 코드 도메인 | 고유 코드로 사용자 구분 | CD-NOMA |
| SIC | 강한 신호부터 순차 제거 | 핵심 수신 기술 |
| 스펙트럼 효율 | OMA 대비 30% 이상 향상 | 동일 자원 다중 사용 |
| 대규모 연결 | IoT 대량 기기 수용 | mMTC 지원 |
| 저지연 | 스케줄링 감소 | URLLC 지원 |
