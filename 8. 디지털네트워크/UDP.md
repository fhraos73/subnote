#디지털네트워크 #UDP #전송계층 #필수 #디지털네트워크/TCP/UDP
## 정의
비연결형 경량 전송 프로토콜, UDP
- User Datagram Protocol, OSI 4계층에서 동작하는 비연결형 프로토콜
- 연결 설정 없이 데이터그램 단위로 빠른 전송, 신뢰성보다 속도 우선
## 키워드
* 비연결형, 데이터그램, Best-Effort, 체크섬, 실시간, 브로드캐스트, 멀티캐스트
## 암기법
* UDP 특징: 비비순무(비연결, 비신뢰, 순서미보장, 무흐름제어)
* UDP 용도: 디브이스(DNS, 브로드캐스트, VoIP, 스트리밍)
## TCP vs UDP 비교
| 구분 | TCP | UDP |
|------|-----|-----|
| 연결 방식 | 연결지향 | 비연결 |
| 신뢰성 | 보장 | 미보장 |
| 순서 보장 | 보장 | 미보장 |
| 흐름/혼잡 제어 | 있음 | 없음 |
| 헤더 크기 | 20~60 bytes | 8 bytes |
| 속도 | 느림 | 빠름 |
| 용도 | HTTP, FTP, SMTP | DNS, VoIP, 스트리밍 |
## UDP 헤더 구조
| 필드 | 크기 | 설명 |
|------|------|------|
| Source Port | 16bit | 송신 포트 번호 |
| Destination Port | 16bit | 수신 포트 번호 |
| Length | 16bit | UDP 데이터그램 길이 |
| Checksum | 16bit | 오류 검출 (선택) |
