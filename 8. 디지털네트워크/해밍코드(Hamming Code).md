#디지털네트워크 #오류정정 #FEC #옵션
## 정의
1비트 오류 정정 코드, 해밍코드
- 패리티 비트를 이용하여 1비트 오류를 검출하고 정정할 수 있는 선형 오류 정정 코드
- Richard Hamming이 개발, 2비트 오류 검출 및 1비트 오류 정정 가능
## 키워드
* 패리티 비트, 해밍 거리, SEC-DED, 신드롬(Syndrome), 2^r ≥ m + r + 1
## 암기법
* "패해신정" - 패리티, 해밍거리, 신드롬, 정정
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
|------|------|------|
| 패리티 비트 | 2^n 위치에 삽입 (1,2,4,8...) | 오류 위치 식별 |
| 해밍 거리 | 두 코드워드 간 다른 비트 수 | 최소 3 필요 |
| 신드롬 | 오류 위치를 나타내는 값 | 0이면 오류 없음 |
| SEC-DED | Single Error Correction, Double Error Detection | 추가 패리티 |
### 해밍코드 계산
| 데이터 비트(m) | 패리티 비트(r) | 총 비트 | 코드 |
|----------------|----------------|---------|------|
| 4 | 3 | 7 | (7,4) |
| 8 | 4 | 12 | (12,8) |
| 11 | 4 | 15 | (15,11) |
| 26 | 5 | 31 | (31,26) |
## 구성도
```
┌─────────────────────────────────────────────────────────────┐
│                   해밍코드 (7,4) 구조                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  비트 위치:  1   2   3   4   5   6   7                      │
│             P1  P2  D1  P4  D2  D3  D4                      │
│             ↓   ↓       ↓                                   │
│           패리티     패리티                                  │
│                                                             │
│  [패리티 비트 검사 범위]                                     │
│  P1(위치1): 1,3,5,7 검사 (이진수 끝자리 1)                   │
│  P2(위치2): 2,3,6,7 검사 (이진수 둘째자리 1)                 │
│  P4(위치4): 4,5,6,7 검사 (이진수 셋째자리 1)                 │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                   오류 정정 과정                             │
│                                                             │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐              │
│  │ 수신 코드 │ →  │신드롬 계산│ →  │오류 위치 │              │
│  │          │    │ S=P1P2P4 │    │  식별    │              │
│  └──────────┘    └──────────┘    └────┬─────┘              │
│                                       │                     │
│                    ┌──────────────────┴──────────────┐      │
│                    ↓                                 ↓      │
│             S = 0 (오류 없음)              S ≠ 0 (S번째 비트 반전) │
│                                                             │
│  예시: S = 101(이진) = 5 → 5번째 비트에 오류                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
