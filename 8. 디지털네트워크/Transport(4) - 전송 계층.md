#디지털네트워크 #전송계층 #OSI #옵션 #디지털네트워크/OSI/프로토콜
## 정의
종단간 신뢰성 있는 데이터 전송 계층, Transport Layer
- OSI 7계층 중 4계층으로 End-to-End 간 데이터 전송 담당
- 포트 번호를 통한 프로세스 간 통신 및 흐름/오류 제어 수행
## 키워드
* TCP, UDP, 포트번호, 세그먼트, 흐름제어, 오류제어, 혼잡제어, 소켓
## 암기법
* 전송계층 기능: 흐오혼다(흐름제어, 오류제어, 혼잡제어, 다중화)
* 포트번호: 웰노다(Well-known 0~1023, Registered 1024~49151, Dynamic 49152~65535)
## 구성요소
| 구분 | 설명 | 비고 |
|------|------|------|
| TCP | 연결지향, 신뢰성 보장 | 3-Way Handshake |
| UDP | 비연결, 빠른 전송 | 실시간 서비스 |
| 포트번호 | 프로세스 식별자 | 0~65535 |
| 세그먼트 | 전송 계층 PDU | 헤더+데이터 |
| 소켓 | IP+Port 조합 | 통신 종단점 |
## 주요 기능
| 기능 | 설명 | 기법 |
|------|------|------|
| 흐름제어 | 송수신 속도 조절 | 슬라이딩 윈도우 |
| 오류제어 | 손실/손상 데이터 복구 | ARQ, 재전송 |
| 혼잡제어 | 네트워크 혼잡 방지 | Slow Start, AIMD |
| 다중화 | 여러 프로세스 동시 통신 | 포트번호 활용 |
| 분할/재조립 | 데이터 분할 및 순서 조립 | 시퀀스 번호 |
