#확률통계 #확률분포 #이항분포 #필수 #확률통계/확률분포
## 정의
성공/실패 두 가지 결과의 반복 시행 분포, Binomial Distribution
- 실패/성공처럼 단 2개의 확률변수만이 존재하는 분포
- n번 독립 시행에서 성공 횟수 k의 확률분포
## 키워드
* n(시행횟수), p(성공확률), 독립시행, 베르누이 시행, 이산확률분포
## 암기법
* "엔피케이" - n(시행수), p(확률), k(성공횟수)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 조건 | 독립 시행, 성공확률 일정 | 베르누이 시행 반복 |
| 평균 | E(X) = np | 기대값 |
| 분산 | Var(X) = np(1-p) | 산포도 |
| 활용 | 불량률, 합격률 계산 | 품질관리 |
