#확률통계 #추론통계 #추정 #필수 #확률통계/추론통계
## 정의
표본으로 모집단의 모수를 추정하는 기법, Statistical Estimation
- 전수조사가 불가능한 모집단의 통계량 구하기 위해 표본을 추출하여 모집단의 통계량을 추정하는 통계 기법
## 키워드
* 점추정, 구간추정, 신뢰구간, 신뢰도, 표본평균, 표본분산, 추정량
## 암기법
* "점구신오" - 점추정, 구간추정, 신뢰도, 오차율
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 점추정 | 하나의 값으로 모수 추정 | 표본평균, 표본분산 |
| 구간추정 | 구간으로 모수 추정 | 신뢰구간 사용 |
| 추정량 조건 | 불편성, 효율성, 일치성, 충분성 | 4가지 조건 |
| 신뢰구간 | P(L ≤ μ ≤ U) = 1-α | 95%, 99% 등 |
