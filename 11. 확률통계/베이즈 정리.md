## 정의
사전확률과 우도로 사후확률을 구하는 확률 추론 방법, Bayes' Theorem
- P(A|B) = P(B|A)P(A) / P(B)
- 어떤 증거 B가 나타났을 때, 원인 A가 실제로 발생했을 확률 추정
## 키워드
* 사전확률(Prior), 사후확률(Posterior), 우도(Likelihood), 증거(Evidence), 나이브 베이즈
## 암기법
* "아비 = 아바/비" - P(A|B) = P(B|A)P(A) / P(B)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| P(A) | 사전확률(Prior) | 기경험 확률 |
| P(B\|A) | 우도(Likelihood) | 가능성 |
| P(B) | 증거(Evidence) | 전체 확률 |
| P(A\|B) | 사후확률(Posterior) | 결과 확률 |
