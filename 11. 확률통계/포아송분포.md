#확률통계 #확률분포 #포아송 #필수 #확률통계/확률분포
## 정의
단위 시간/공간에서 사건 발생 횟수의 분포, Poisson Distribution
- 무작위로 일어나는 사건의 발생횟수에 적용되는 분포
- 예: 1분 동안 콜센터에 걸려오는 전화 수
## 키워드
* λ(평균 발생률), 이산확률분포, 독립사건, 이항분포 근사
## 암기법
* "람다" - λ(평균 발생률)가 핵심 파라미터
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 조건 | 독립 사건, 일정한 발생률 | 희귀 사건 |
| 평균 | E(X) = λ | 기대값=분산 |
| 분산 | Var(X) = λ | 평균=분산 특성 |
| 활용 | 서버 요청 수, 사고 발생 수 | 대기행렬이론 |
