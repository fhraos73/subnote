## 정의
표본 크기가 충분히 크면 표본평균이 정규분포에 수렴하는 원리, CLT
- 모집단으로부터 추출된 표본의 크기 n이 충분히 크다면(30 이상) 표본 평균들의 분포는 모집단 분포와 상관없이 정규분포를 따름
## 키워드
* n≥30, 표본평균, 정규분포 수렴, 모집단 분포 무관, 표본분포
## 암기법
* "삼정수" - 30이상이면 정규분포로 수렴
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 조건 | 표본 크기 n ≥ 30 | 충분히 큰 표본 |
| 결과 | 표본평균 ~ N(μ, σ²/n) | 정규분포 수렴 |
| 의의 | 모집단 분포 몰라도 추론 가능 | 추론통계 근간 |
