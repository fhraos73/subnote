#확률통계 #확률분포 #베르누이 #필수 #확률통계/확률분포
## 정의
성공/실패 두 가지 결과만 갖는 단일 시행의 분포, Bernoulli Distribution
- 한 번의 시행에서 성공(1) 또는 실패(0)의 두 가지 결과만 존재하는 확률분포
## 키워드
* 성공확률 p, 실패확률 (1-p), 이산확률분포, 이항분포의 기본
## 암기법
* "영일피" - 0(실패) 또는 1(성공), p(성공확률)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 시행 | 단 1회 독립시행 | 성공/실패만 존재 |
| 평균 | E(X) = p | 성공확률 |
| 분산 | Var(X) = p(1-p) | 최대 p=0.5 |
| 관계 | n번 반복 → 이항분포 | 이항분포의 기본 |
