#확률통계 #분석 #다중공선성 #필수 #확률통계/분석
## 정의
독립변수 간 강한 상관관계로 인한 회귀분석 문제, Multicollinearity
- 다중 회귀분석에서 사용된 모형의 일부 독립변수가 다른 독립변수와 상관 정도가 높아 데이터 분석 시 부정적인 영향을 미치는 현상
## 키워드
* VIF(분산팽창인자), 상관계수, 독립변수 간 상관, 결정계수, 회귀계수 불안정
## 암기법
* "비아이에프" - VIF > 10이면 다중공선성 의심
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 원인 | 독립변수 간 높은 상관관계 | r > 0.9 |
| 진단 | VIF(분산팽창인자) | VIF > 10 의심 |
| 영향 | 회귀계수 불안정, 해석 어려움 | 분산 증가 |
| 해결 | 변수 제거, PCA, 릿지 회귀 | 차원 축소 |
