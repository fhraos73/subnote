#확률통계 #추론통계 #p-value #필수 #확률통계/추론통계
## 정의
관찰 데이터가 귀무가설과 양립하는 정도, p-value
- 관찰된 데이터가 귀무가설과 양립하는 정도를 0에서 1 사이의 수치로 표현한 것
- p-value < α이면 귀무가설 기각
## 키워드
* 유의수준(α), 귀무가설, 기각, 채택, 통계적 유의성
## 암기법
* "피알파" - p < α이면 기각, p ≥ α이면 채택
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| p < 0.01 | 매우 유의 | 강한 증거 |
| p < 0.05 | 유의 | 일반적 기준 |
| p ≥ 0.05 | 유의하지 않음 | 귀무가설 채택 |
