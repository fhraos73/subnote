#확률통계 #검정 #z검정 #필수 #확률통계/추론통계
## 정의
모집단의 평균/표준편차가 알려져 있을 때 표본 평균을 검정하는 기법, z-test
- 모집단의 평균과 표준편차가 알려져 있을 때, 새로 조사된 표본의 평균이 모집단의 평균과 같은지를 추정하는 검정
## 키워드
* 표준정규분포, 대표본(n≥30), 모분산 기지, Z-통계량
## 암기법
* "모알대" - 모분산 알려진 대표본 검정
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 조건 | 모분산 기지 또는 n≥30 | 정규분포 가정 |
| 통계량 | Z = (x̄ - μ) / (σ/√n) | 표준화 |
| 기각역 | Z > Zα (단측), \|Z\| > Zα/2 (양측) | 유의수준 기준 |
