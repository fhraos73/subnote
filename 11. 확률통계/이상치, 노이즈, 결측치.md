#확률통계 #데이터전처리 #이상치 #필수 #확률통계/데이터전처리
## 정의
데이터 품질을 저해하는 비정상 데이터 유형, Outlier/Noise/Missing Value
- 이상치: 기준에서 벗어난 값, 특이 패턴 탐지 대상
- 노이즈: 불규칙하고 무의미한 오류 값
- 결측치: 수집되지 않거나 누락된 데이터
## 키워드
* Z-Score, IQR, Boxplot, KNN, DBSCAN, 평균대체, 삭제법, 보간법
## 암기법
* "이노결" - 이상치(탐지), 노이즈(제거), 결측치(대체)
## 구성요소/특징/유형
| 구분 | 설명 | 처리방법 |
| ---- | ---- | ---- |
| 이상치 | 데이터 분포에서 크게 벗어난 값 | Z-Score, IQR, DBSCAN |
| 노이즈 | 불규칙 오류 값, 센서 오작동 등 | 필터링, 평활화 |
| 결측치 | 수집 누락 데이터 | 삭제, 평균대체, 보간 |
