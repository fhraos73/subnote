#확률통계 #분석 #회귀분석 #필수 #확률통계/분석
## 정의
독립변수와 종속변수 간 함수적 관계를 통계적으로 추정하는 방법, Regression Analysis
- 독립변수들과 종속변수 간에 존재하는 관련성을 분석하기 위하여 관측된 자료에서 이들 간의 함수적 관계를 통계적으로 추정하는 방법
## 키워드
* 독립변수(X), 종속변수(Y), 회귀계수, 최소자승법(OLS), 결정계수(R²), 잔차
## 암기법
* "독단다 예선로" - 독립변수 수(단순/다중), 예측모델(선형/로지스틱)
* "독종회최방" - 독립변수, 종속변수, 회귀계수, 최소자승법, 회귀방정식
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 단순회귀 | 독립변수 1개 | Y = a + bX |
| 다중회귀 | 독립변수 2개 이상 | Y = a + b₁X₁ + b₂X₂ |
| 선형회귀 | 연속값 예측 | 최소제곱법(OLS) |
| 로지스틱회귀 | 분류(범주 예측) | 시그모이드, MLE |
