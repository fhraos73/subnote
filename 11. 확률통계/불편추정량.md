#확률통계 #추론통계 #불편추정량 #필수 #확률통계/추론통계
## 정의
편의(Bias)가 없는 추정량, Unbiased Estimator
- 추정량의 기대값과 실제 모수와의 차이인 편의(Bias)가 없는 추정량으로, 기대값과 모수가 동일한 추정량
## 키워드
* 추정량, 편의(Bias), 표본평균, 표본분산, 자유도(n-1), 추정통계
## 암기법
* "불효일충" - 불편성, 효율성, 일치성, 충분성 (좋은 추정량의 4조건)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 불편성 | E(θ̂) = θ, 편의 없음 | 핵심 조건 |
| 효율성 | 분산이 가장 작은 추정량 | MVUE |
| 일치성 | n→∞ 시 모수에 수렴 | 대표본 특성 |
| 충분성 | 모수에 대한 모든 정보 포함 | 정보 손실 無 |
| 자유도 | n-1로 나누는 이유 | 표본분산 불편추정 |
