#확률통계 #추론통계 #가설검정 #필수 #확률통계/추론통계
## 정의
모집단에 대한 가설의 진위를 표본으로 판정하는 과정, Hypothesis Test
- 모집단 모수의 값을 설정하고(가설 설정), 표본 통계치를 통해 확률적으로 진위를 판정하는 과정
## 키워드
* 귀무가설(H₀), 대립가설(H₁), 유의수준(α), 기각역, 검정통계량, 1종오류, 2종오류
## 암기법
* "가대귀 유기 통방-양단" - 가설, 대립가설, 귀무가설, 유의수준, 기각역, 통계량, 방법-양측/단측
* "명가특경" - 명료성, 가치중립성, 특정화, 경험적 검증 가능성
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 귀무가설(H₀) | 기각하고 싶은 주장 | 귀기모 |
| 대립가설(H₁) | 검증하고자 하는 주장 | 대검모 |
| 유의수준(α) | 귀무가설 기각 확률 크기 | 0.05, 0.01 |
| 1종 오류 | H₀ 참인데 기각 | α |
| 2종 오류 | H₀ 거짓인데 채택 | β |
| 검정 방법 | 양측/단측 검정 | 방향성 여부 |
