## 정의
두 집단 이하의 표본 평균 간 차이를 검정하는 기법, t-test
- 모집단의 분산이 알려지지 않고 표본 크기가 작을 때(n<30), 표본 평균 간 차이를 검정
## 키워드
* t-분포, 자유도, 독립표본, 대응표본, 단일표본, 소표본
## 암기법
* "독대단" - 독립표본, 대응표본, 단일표본 t-검정
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 단일표본 | 하나의 표본과 모집단 평균 비교 | μ vs x̄ |
| 독립표본 | 두 독립 집단의 평균 비교 | x̄₁ vs x̄₂ |
| 대응표본 | 동일 집단의 전/후 비교 | 짝비교 |
