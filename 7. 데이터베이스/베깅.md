## 정의
Bootstrap Aggregating
- 원본 데이터에서 복원 추출로 여러 샘플을 생성하여 병렬 학습 후 결과를 집계하는 앙상블 기법
- 분산(Variance)을 줄여 과적합을 방지하고 모델 안정성을 향상시키는 방법
## 키워드
* Bootstrap Sampling, Aggregating, 복원 추출
* 분산 감소, 과적합 방지, 병렬 학습
* 다수결 투표(분류), 평균(회귀), 랜덤 포레스트
## 암기법
* "배깅은 B.A" - Bootstrap + Aggregating
* "복병집분" - 복원추출, 병렬학습, 집계, 분산감소
## 구성요소/특징/유형
### 배깅의 핵심 구성
| 구성요소 | 설명 |
|---------|------|
| Bootstrap | 원본 데이터에서 중복 허용 복원 추출 |
| Base Learner | 동일한 알고리즘의 개별 학습기 |
| Aggregating | 분류: 다수결 투표, 회귀: 평균값 |
### 배깅의 특징
- **병렬 처리**: 각 모델 독립적 학습 가능
- **분산 감소**: 여러 모델 결합으로 예측 안정화
- **과적합 완화**: 다양한 샘플로 일반화 성능 향상
- **강건성**: 이상치, 노이즈에 덜 민감
### 배깅 vs 부스팅
| 구분 | 배깅 | 부스팅 |
|------|------|--------|
| 학습 방식 | 병렬 | 순차 |
| 목표 | 분산 감소 | 편향 감소 |
| 가중치 | 동일 | 오분류 샘플 가중 |
| 과적합 | 낮음 | 중간~높음 |
| 대표 알고리즘 | Random Forest | AdaBoost, XGBoost |
### 배깅의 한계
- 높은 편향 모델에서는 효과 제한
- 저분산 모델(선형회귀 등)에서 개선 미미
- 다수의 모델 학습으로 계산 비용 증가
## 구성도
```
┌─────────────────────────────────────────────────────┐
│                  배깅(Bagging) 프로세스              │
├─────────────────────────────────────────────────────┤
│         ┌─────────────────────┐                     │
│         │   원본 데이터셋 D    │                     │
│         └──────────┬──────────┘                     │
│                    │ Bootstrap Sampling             │
│         ┌──────────┼──────────┐                     │
│         ▼          ▼          ▼                     │
│    ┌────────┐ ┌────────┐ ┌────────┐                │
│    │샘플 D1 │ │샘플 D2 │ │샘플 Dn │  (복원 추출)    │
│    └───┬────┘ └───┬────┘ └───┬────┘                │
│        │          │          │                      │
│        ▼          ▼          ▼      (병렬 학습)     │
│    ┌────────┐ ┌────────┐ ┌────────┐                │
│    │모델 M1 │ │모델 M2 │ │모델 Mn │                │
│    └───┬────┘ └───┬────┘ └───┬────┘                │
│        │          │          │                      │
│        ▼          ▼          ▼                      │
│    ┌────────┐ ┌────────┐ ┌────────┐                │
│    │예측 P1 │ │예측 P2 │ │예측 Pn │                │
│    └───┬────┘ └───┬────┘ └───┬────┘                │
│        └──────────┼──────────┘                      │
│                   ▼  Aggregating                    │
│         ┌─────────────────────┐                     │
│         │     최종 예측       │                     │
│         │ 분류: 다수결 투표   │                     │
│         │ 회귀: 평균값        │                     │
│         └─────────────────────┘                     │
└─────────────────────────────────────────────────────┘
```
