#데이터베이스 #데이터분석 #앙상블 #머신러닝 #필수
## 정의
순차적 약한 학습기 결합
- 약한 학습기(Weak Learner)를 순차적으로 학습시켜 이전 모델의 오류를 보완하는 앙상블 기법
- 오분류 샘플에 가중치를 부여하여 편향(Bias)을 줄이고 예측 정확도를 향상
## 키워드
* Weak Learner, 순차 학습, 가중치 업데이트
* AdaBoost, Gradient Boosting, XGBoost, LightGBM
* 편향 감소, 오분류 보정, 잔차(Residual)
## 암기법
* "부스팅은 순편가" - 순차학습, 편향감소, 가중치부여
* "아그엑라" - AdaBoost, Gradient Boosting, XGBoost, LightGBM
## 구성요소/특징/유형
### 부스팅의 핵심 원리
| 단계 | 설명 |
|------|------|
| 1단계 | 첫 번째 약한 학습기 훈련 |
| 2단계 | 오분류 샘플에 높은 가중치 부여 |
| 3단계 | 다음 학습기가 가중치 반영하여 학습 |
| 4단계 | 모든 학습기 예측을 가중 결합 |
### 주요 부스팅 알고리즘
| 알고리즘 | 특징 | 장점 | 단점 |
|---------|------|------|------|
| AdaBoost | 오분류 샘플 가중치 증가 | 단순, 구현 용이 | 노이즈에 민감 |
| Gradient Boosting | 잔차(Residual) 학습 | 유연성, 정확도 | 느린 속도 |
| XGBoost | 정규화 + 병렬처리 | 빠른 속도, 과적합 방지 | 튜닝 복잡 |
| LightGBM | Leaf-wise 성장 | 대용량 데이터 효율 | 과적합 위험 |
### 부스팅 vs 배깅
| 구분 | 부스팅 | 배깅 |
|------|--------|------|
| 학습 방식 | 순차적 | 병렬적 |
| 목표 | 편향 감소 | 분산 감소 |
| 샘플링 | 가중치 기반 | 복원 추출 |
| 과적합 위험 | 중간~높음 | 낮음 |
| 속도 | 느림 | 빠름 |
### XGBoost 핵심 특징
- **정규화**: L1/L2 정규화로 과적합 방지
- **병렬 처리**: 트리 구축 시 병렬화
- **결측값 처리**: 자동 결측값 처리
- **조기 종료**: Early Stopping 지원
## 구성도
```
┌─────────────────────────────────────────────────────┐
│                부스팅(Boosting) 프로세스             │
├─────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────┐   │
│  │              원본 데이터셋                   │   │
│  └──────────────────┬──────────────────────────┘   │
│                     ▼                               │
│  ┌─────────────────────────────────────────────┐   │
│  │  Round 1: 약한 학습기 h1 훈련               │   │
│  │  → 오분류 샘플 식별                         │   │
│  └──────────────────┬──────────────────────────┘   │
│                     ▼ 가중치 업데이트              │
│  ┌─────────────────────────────────────────────┐   │
│  │  Round 2: 약한 학습기 h2 훈련               │   │
│  │  → 이전 오류에 집중                         │   │
│  └──────────────────┬──────────────────────────┘   │
│                     ▼ 가중치 업데이트              │
│  ┌─────────────────────────────────────────────┐   │
│  │  Round n: 약한 학습기 hn 훈련               │   │
│  └──────────────────┬──────────────────────────┘   │
│                     ▼                               │
│  ┌─────────────────────────────────────────────┐   │
│  │        가중 결합 (Weighted Combination)      │   │
│  │   H(x) = Σ αi × hi(x)                       │   │
│  │   (αi: 각 학습기의 가중치)                  │   │
│  └──────────────────┬──────────────────────────┘   │
│                     ▼                               │
│  ┌─────────────────────────────────────────────┐   │
│  │              최종 강한 학습기                │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```
