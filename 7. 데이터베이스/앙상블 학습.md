#데이터베이스 #데이터분석 #앙상블 #머신러닝 #필수
## 정의
다중 모델 결합 학습법
- 여러 개의 학습 모델을 결합하여 단일 모델보다 우수한 예측 성능을 달성하는 기법
- 배깅, 부스팅, 스태킹 세 가지 주요 방식으로 분류
## 키워드
* Bagging, Boosting, Stacking, Voting
* 분산 감소, 편향 감소, 메타 모델
* 약한 학습기(Weak Learner), 강한 학습기(Strong Learner)
## 암기법
* "앙상블은 배부스" - 배깅, 부스팅, 스태킹
* "배병분 부순편 스메타" - 배깅(병렬/분산), 부스팅(순차/편향), 스태킹(메타모델)
## 구성요소/특징/유형
### 앙상블 기법 비교
| 구분 | 배깅(Bagging) | 부스팅(Boosting) | 스태킹(Stacking) |
|------|--------------|-----------------|-----------------|
| 학습 방식 | 병렬 | 순차 | 병렬 + 메타 |
| 목표 | 분산 감소 | 편향 감소 | 편향 감소 |
| 알고리즘 | 동일 | 동일 | 다양 |
| 과적합 위험 | 낮음 | 중간 | 높음 |
| 계산 속도 | 빠름 | 느림 | 느림 |
| 대표 알고리즘 | Random Forest | XGBoost, AdaBoost | Stacked Generalization |
### 배깅(Bagging)
- **원리**: Bootstrap 샘플링 → 병렬 학습 → 집계(Aggregating)
- **특징**: 복원 추출, 동일 알고리즘, 분산 감소
- **집계**: 분류(다수결 투표), 회귀(평균)
- **대표**: Random Forest
### 부스팅(Boosting)
- **원리**: 순차 학습 → 오분류 가중치 부여 → 가중 결합
- **특징**: 약한 학습기 연속 결합, 편향 감소
- **대표**: AdaBoost, Gradient Boosting, XGBoost, LightGBM
### 스태킹(Stacking)
- **원리**: 기반 모델 예측 → 메타 모델 학습
- **특징**: 다양한 알고리즘 조합, 2단계 학습
- **구성**: Level 0(기반 모델들) + Level 1(메타 모델)
- **주의**: 과적합 방지를 위해 교차검증 필수
### 보팅(Voting)
- **Hard Voting**: 다수결 투표
- **Soft Voting**: 확률 평균 기반 투표
