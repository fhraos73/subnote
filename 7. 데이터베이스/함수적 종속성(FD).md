#데이터베이스 #함수적종속성 #정규화 #필수
## 정의
속성 간 결정 관계, FD
- 릴레이션에서 속성 X의 값이 속성 Y의 값을 유일하게 결정하는 제약 관계(X→Y)
- X를 결정자(Determinant), Y를 종속자(Dependent)라 하며 정규화의 이론적 기반
## 키워드
* 결정자(Determinant): 다른 속성을 결정하는 속성
* 종속자(Dependent): 결정자에 의해 결정되는 속성
* 완전함수종속(Full FD): 기본키 전체에 종속
* 부분함수종속(Partial FD): 기본키 일부에 종속
* 이행함수종속(Transitive FD): X→Y, Y→Z ⇒ X→Z
## 암기법
* "완부이" - 완전/부분/이행 함수종속
* "결종" - 결정자가 종속자를 결정
* "2부3이" - 2NF는 부분종속 제거, 3NF는 이행종속 제거
## 구성요소/특징/유형
### 함수적 종속성 유형
| 유형 | 정의 | 예시 | 정규화 |
|------|------|------|--------|
| 완전함수종속 | 기본키 전체에 종속 | {학번,과목}→성적 | 정규화 완료 |
| 부분함수종속 | 기본키 일부에 종속 | {학번,과목}에서 학번→학과 | 2NF로 제거 |
| 이행함수종속 | X→Y, Y→Z 관계 | 학번→학과, 학과→학과장 | 3NF로 제거 |
### Armstrong 공리
| 공리 | 설명 |
|------|------|
| 반사성(Reflexivity) | Y⊆X이면 X→Y |
| 첨가성(Augmentation) | X→Y이면 XZ→YZ |
| 이행성(Transitivity) | X→Y, Y→Z이면 X→Z |
### 추가 규칙
- 분해(Decomposition): X→YZ이면 X→Y, X→Z
- 합집합(Union): X→Y, X→Z이면 X→YZ
- 의사이행(Pseudotransitivity): X→Y, WY→Z이면 WX→Z
## 구성도
```
┌─────────────────────────────────────────────────────────────┐
│                    함수적 종속성 유형                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 완전함수종속 (Full FD)                                   │
│     ┌─────────────┐                                         │
│     │ 학번 + 과목  │ ──────→ 성적                            │
│     └─────────────┘   (전체 키에 종속)                       │
│                                                             │
│  2. 부분함수종속 (Partial FD) - 2NF에서 제거                  │
│     ┌─────────────┐                                         │
│     │ 학번 + 과목  │                                         │
│     └──────┬──────┘                                         │
│            │                                                │
│         학번 ──────→ 학과 (일부 키에 종속)                    │
│                                                             │
│  3. 이행함수종속 (Transitive FD) - 3NF에서 제거               │
│     학번 ──→ 학과 ──→ 학과장                                 │
│       └────────────────→ (이행적 종속)                       │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                  Armstrong 공리 체계                         │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐                │
│  │ 반사성   │   │ 첨가성   │   │ 이행성   │                │
│  │ Y⊆X→X→Y │   │ X→Y      │   │ X→Y,Y→Z  │                │
│  │          │   │ ⇒XZ→YZ  │   │ ⇒X→Z    │                │
│  └──────────┘   └──────────┘   └──────────┘                │
└─────────────────────────────────────────────────────────────┘
```
