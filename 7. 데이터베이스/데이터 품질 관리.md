#데이터베이스 #데이터관리 #품질 #필수 #데이터베이스/데이터관리
## 정의
데이터 신뢰성 확보를 위한, 데이터 품질 관리
- 데이터의 정확성, 완전성, 일관성, 적시성을 측정하고 개선하는 활동
- DQM(Data Quality Management)을 통해 데이터 가치를 극대화하는 체계
## 키워드
* 정확성, 완전성, 일관성, 적시성, 유일성, 유효성, DQI
## 암기법
* "정완일적유유" - 정확성, 완전성, 일관성, 적시성, 유일성, 유효성
## 품질 지표 (6대 품질 기준)
| 품질 기준 | 설명 | 측정 방법 |
|-----------|------|-----------|
| 정확성 | 실제 값과 일치 | 오류율 측정 |
| 완전성 | 필수 데이터 존재 | NULL 비율 |
| 일관성 | 데이터 간 모순 없음 | 참조 무결성 |
| 적시성 | 최신 데이터 유지 | 갱신 주기 |
| 유일성 | 중복 데이터 없음 | 중복률 측정 |
| 유효성 | 도메인 규칙 준수 | 범위 검증 |
