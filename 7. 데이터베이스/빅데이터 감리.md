#데이터베이스 #빅데이터 #감리 #정보화사업 #필수 #SW공학/정보시스템_감리 #데이터베이스/빅데이터/분석
## 정의
빅데이터 사업 품질 검증
- 빅데이터 관련 정보화 사업의 품질과 적정성을 독립적으로 검증하는 감리 활동
- 정보시스템 감리기준에 빅데이터 특성을 반영한 점검항목 적용
## 키워드
* 정보시스템 감리기준, 빅데이터 품질, 데이터 거버넌스
* 수집/저장/처리/분석/활용, 개인정보보호
* 분석 모델 검증, 데이터 정합성, 보안성
## 암기법
* "수저처분활보" - 수집, 저장, 처리, 분석, 활용, 보안
* "품적기보" - 품질, 적정성, 기술성, 보안성
## 구성요소/특징/유형
### 빅데이터 감리 법적 근거
| 근거 | 내용 |
|------|------|
| 전자정부법 | 정보시스템 감리 의무화 |
| 정보시스템 감리기준 | 행정안전부 고시 제2021-4호 |
| 개인정보보호법 | 빅데이터 개인정보 처리 기준 |
| 데이터기본법 | 데이터 품질관리 기준 |
### 빅데이터 감리 점검 영역
| 영역 | 점검항목 |
|------|---------|
| 데이터 수집 | 수집 적법성, 동의 절차, 수집 범위 적정성 |
| 데이터 저장 | 저장소 설계, 확장성, 백업/복구 체계 |
| 데이터 처리 | 처리 성능, 분산처리 적정성, ETL 품질 |
| 데이터 분석 | 분석 모델 적합성, 알고리즘 검증, 정확도 |
| 데이터 활용 | 시각화, 의사결정 지원, 서비스 연계 |
| 보안/개인정보 | 비식별화, 접근통제, 암호화 |
### 단계별 감리 점검
| 단계 | 주요 점검사항 |
|------|-------------|
| 요구분석 | 빅데이터 요구사항 명확성, 분석 목표 적정성 |
| 설계 | 아키텍처 적정성, 데이터 모델 설계, 보안 설계 |
| 구현 | 구현 적합성, 코드 품질, 테스트 충분성 |
| 시험 | 성능 시험, 분석 결과 검증, 보안 취약점 점검 |
### 빅데이터 특화 점검항목
| 항목 | 점검 내용 |
|------|----------|
| 데이터 품질 | 정확성, 완전성, 일관성, 적시성 |
| 분석 모델 | 모델 적합성, 과적합 여부, 재현성 |
| 확장성 | 데이터 증가 대응, 처리 성능 확장 |
| 개인정보 | 비식별화 적정성, 재식별 위험 평가 |
| 거버넌스 | 데이터 관리 체계, 메타데이터 관리 |
### 감리 수행 절차
1. **감리 계획**: 범위, 일정, 투입인력 결정
2. **현황 파악**: 문서 검토, 인터뷰, 시스템 점검
3. **점검 수행**: 점검항목별 적합성 검증
4. **결과 분석**: 문제점 도출, 개선방안 수립
5. **보고서 작성**: 감리 결과 보고서 작성
