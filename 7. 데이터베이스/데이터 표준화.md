#데이터베이스 #데이터관리 #표준화 #필수 #데이터베이스/데이터관리
## 정의
데이터 일관성 확보를 위한, 데이터 표준화
- 조직 내 데이터 요소의 명칭, 정의, 형식, 규칙을 통일하여 관리하는 체계
- 데이터 사전, 명명규칙, 도메인, 코드를 표준화하여 데이터 품질 향상
## 키워드
* 표준 단어, 표준 용어, 표준 도메인, 표준 코드, 데이터 사전
## 암기법
* "단용도코" - 단어, 용어, 도메인, 코드
## 표준화 대상
| 대상 | 설명 | 예시 |
|------|------|------|
| 표준 단어 | 업무에서 사용하는 최소 의미 단위 | 고객, 번호, 일자 |
| 표준 용어 | 단어 조합으로 구성된 컬럼명 | 고객번호, 주문일자 |
| 표준 도메인 | 데이터 타입과 길이 정의 | VARCHAR(20), DATE |
| 표준 코드 | 코드값과 의미 정의 | Y/N, 01/02/03 |
## 구성도
```
┌─────────────────────────────────────────┐
│           데이터 표준 관리 시스템         │
├──────────┬──────────┬──────────┬────────┤
│ 표준단어 │ 표준용어 │ 표준도메인│표준코드│
├──────────┴──────────┴──────────┴────────┤
│              데이터 사전                 │
├─────────────────────────────────────────┤
│           메타데이터 저장소              │
└─────────────────────────────────────────┘
```
## 표준화 절차
```
[현황분석] → [표준정의] → [표준적용] → [표준관리]
 데이터현황    단어/용어     시스템반영    변경관리
 분석/수집    도메인/코드    검증/배포    모니터링
```
