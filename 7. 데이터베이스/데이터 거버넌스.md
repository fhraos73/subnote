#데이터베이스 #데이터관리 #거버넌스 #필수 #데이터베이스/데이터관리
## 정의
데이터 자산의 관리 체계, 데이터 거버넌스
- 조직 내 데이터의 가용성, 유용성, 무결성, 보안을 관리하는 전사적 데이터 관리 체계
- 데이터 정책, 표준, 프로세스를 수립하고 데이터 품질을 보장하는 프레임워크
## 키워드
* 데이터 스튜어드, 메타데이터, 데이터 품질, 마스터 데이터, 데이터 카탈로그
## 암기법
* "정표품보" - 정책, 표준, 품질, 보안
## 구성요소
| 구성요소 | 설명 |
|----------|------|
| 데이터 정책 | 데이터 관리 원칙 및 지침 |
| 데이터 표준 | 명명규칙, 도메인, 코드 표준 |
| 데이터 품질 | 정확성, 완전성, 일관성 관리 |
| 데이터 보안 | 접근권한, 암호화, 감사 |
| 메타데이터 | 데이터에 대한 데이터 관리 |
| 마스터 데이터 | 핵심 기준 데이터 관리 |

## 역할 체계
| 역할 | 책임 |
|------|------|
| CDO(Chief Data Officer) | 전사 데이터 총괄 |
| 데이터 스튜어드 | 도메인별 데이터 관리 |
| 데이터 아키텍트 | 데이터 구조 설계 |
| 데이터 관리자 | 일상적 데이터 운영 |
