#데이터베이스 #데이터분석 #머신러닝 #모델평가 #필수
## 정의
ML 모델 성능 측정 방법
- 머신러닝 분류/회귀 모델의 예측 성능을 정량적으로 측정하는 평가 지표 및 방법론
- 혼동행렬 기반의 정확도, 정밀도, 재현율, F1 스코어 등 활용
## 키워드
* 혼동행렬(Confusion Matrix), TP, TN, FP, FN
* 정확도(Accuracy), 정밀도(Precision), 재현율(Recall)
* F1 Score, ROC-AUC, 교차검증(Cross Validation)
## 암기법
* "혼정정재F" - 혼동행렬, 정확도, 정밀도, 재현율, F1
* "정밀FP 재현FN" - 정밀도는 FP 줄이기, 재현율은 FN 줄이기
## 구성요소/특징/유형
### 혼동행렬(Confusion Matrix)
|  | 예측: Positive | 예측: Negative |
|--|---------------|---------------|
| 실제: Positive | TP (True Positive) | FN (False Negative) |
| 실제: Negative | FP (False Positive) | TN (True Negative) |
### 분류 모델 평가 지표
| 지표 | 공식 | 의미 | 활용 |
|------|------|------|------|
| 정확도 | (TP+TN)/(TP+TN+FP+FN) | 전체 정답 비율 | 균형 데이터 |
| 정밀도 | TP/(TP+FP) | 양성 예측 중 실제 양성 | FP 최소화 (스팸필터) |
| 재현율 | TP/(TP+FN) | 실제 양성 중 예측 양성 | FN 최소화 (암진단) |
| F1 Score | 2×(P×R)/(P+R) | 정밀도-재현율 조화평균 | 불균형 데이터 |
| 특이도 | TN/(TN+FP) | 실제 음성 중 예측 음성 | 음성 정확도 |
### 회귀 모델 평가 지표
| 지표 | 설명 |
|------|------|
| MSE | 평균 제곱 오차 |
| RMSE | 평균 제곱근 오차 |
| MAE | 평균 절대 오차 |
| R² | 결정계수 (설명력) |
### ROC-AUC
- **ROC 곡선**: FPR(1-특이도) vs TPR(재현율) 그래프
- **AUC**: ROC 곡선 아래 면적 (0.5~1.0)
- **해석**: AUC 1.0 = 완벽, 0.5 = 랜덤 분류
### 교차검증(Cross Validation)
| 방법 | 설명 |
|------|------|
| Hold-out | 학습/검증 데이터 분리 |
| K-Fold | K개 폴드로 분할, K번 검증 |
| Stratified K-Fold | 클래스 비율 유지 K-Fold |
| LOOCV | Leave-One-Out, N번 검증 |
### 평가 지표 선택 기준
- **균형 데이터**: 정확도 사용 가능
- **불균형 데이터**: F1 Score, AUC 권장
- **FP 비용 높음**: 정밀도 중시 (스팸 필터)
- **FN 비용 높음**: 재현율 중시 (암 진단)
