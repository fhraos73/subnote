#데이터베이스 #정규화 #2NF #필수 #데이터베이스/정규화
## 정의
부분함수종속 제거, 2NF
- 1NF를 만족하면서 기본키가 아닌 모든 속성이 기본키에 완전함수종속하는 정규화 단계
- 복합키의 일부에만 종속되는 부분함수종속을 제거하여 테이블 분리
## 키워드
* 부분함수종속(Partial FD): 복합키의 일부에만 종속
* 완전함수종속(Full FD): 기본키 전체에 종속
* 복합키(Composite Key): 2개 이상의 속성으로 구성된 키
* 테이블 분해(Decomposition): 종속성에 따른 테이블 분리
## 암기법
* "2부완" - 2NF는 부분종속 제거하여 완전종속만 남김
* "일부제" - 일부 키에 대한 종속 제거
* "복합분" - 복합키 있을 때 분리 필요
## 구성요소/특징/유형
### 2NF 조건
| 조건 | 설명 |
|------|------|
| 1NF 만족 | 모든 속성이 원자값 |
| 완전함수종속 | 모든 비키 속성이 기본키 전체에 종속 |
| 부분함수종속 없음 | 기본키 일부에만 종속되는 속성 없음 |
### 2NF 적용 대상
- 복합키를 가진 테이블에서만 의미 있음
- 단일키 테이블은 1NF 만족 시 자동으로 2NF 만족
### 부분함수종속 식별
| 구분 | 설명 |
|------|------|
| 복합키 확인 | 기본키가 2개 이상 속성으로 구성 |
| 종속성 분석 | 각 비키 속성이 키의 어느 부분에 종속되는지 확인 |
| 분리 대상 | 키 일부에만 종속되는 속성 그룹 |
## 구성도
```
┌─────────────────────────────────────────────────────────────┐
│                    2NF 변환 과정                             │
├─────────────────────────────────────────────────────────────┤
│  [2NF 위반 - 부분함수종속 존재]                               │
│  ┌───────────────────────────────────────────────────────┐ │
│  │ 수강 테이블                                            │ │
│  │ 학번(PK) │ 과목코드(PK) │ 성적 │ 학과 │ 과목명         │ │
│  ├───────────────────────────────────────────────────────┤ │
│  │   001   │    C001     │  A  │ 컴공 │  DB            │ │
│  │   001   │    C002     │  B  │ 컴공 │ 네트워크        │ │
│  │   002   │    C001     │  A  │ 전자 │  DB            │ │
│  └───────────────────────────────────────────────────────┘ │
│                                                             │
│  함수적 종속성 분석:                                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ {학번, 과목코드} → 성적     (완전함수종속) ✓          │   │
│  │ {학번} → 학과              (부분함수종속) ✗          │   │
│  │ {과목코드} → 과목명         (부분함수종속) ✗          │   │
│  └─────────────────────────────────────────────────────┘   │
│                          ↓ 2NF 적용                         │
│  ┌─────────────────┐ ┌─────────────────┐ ┌──────────────┐ │
│  │ 학생 테이블      │ │ 과목 테이블      │ │ 수강 테이블   │ │
│  ├─────────────────┤ ├─────────────────┤ ├──────────────┤ │
│  │ 학번(PK)│ 학과  │ │과목코드(PK)│과목명│ │학번│과목코드│성적│
│  │  001  │ 컴공  │ │  C001   │ DB  │ │001│ C001 │ A │
│  │  002  │ 전자  │ │  C002   │네트워크│ │001│ C002 │ B │
│  └─────────────────┘ └─────────────────┘ │002│ C001 │ A │
│                                          └──────────────┘ │
│  결과: 모든 비키 속성이 해당 테이블의 기본키에 완전함수종속     │
└─────────────────────────────────────────────────────────────┘
```
