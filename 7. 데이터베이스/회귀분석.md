## 정의
변수 간 관계 분석 기법
- 독립변수와 종속변수 간의 관계를 수학적 모델로 표현하여 예측하는 통계 분석 기법
- 선형회귀(연속값 예측)와 로지스틱회귀(분류)로 구분
## 키워드
* 선형회귀(Linear Regression), 로지스틱회귀(Logistic Regression)
* 독립변수, 종속변수, 회귀계수, 잔차(Residual)
* 최소제곱법(OLS), 최대가능도추정(MLE), 시그모이드 함수
## 암기법
* "선연로분" - 선형회귀(연속값), 로지스틱회귀(분류)
* "최잔결다" - 최소제곱법, 잔차, 결정계수, 다중공선성
## 구성요소/특징/유형
### 선형회귀 vs 로지스틱회귀
| 구분 | 선형회귀 | 로지스틱회귀 |
|------|---------|-------------|
| 목적 | 연속값 예측 | 분류(범주 예측) |
| 출력 | 실수값 | 확률(0~1) |
| 함수 | 선형 방정식 | 시그모이드 함수 |
| 추정법 | 최소제곱법(OLS) | 최대가능도추정(MLE) |
| 예시 | 주택가격, 매출액 | 스팸분류, 질병진단 |
### 선형회귀 유형
| 유형 | 설명 |
|------|------|
| 단순 선형회귀 | 독립변수 1개, Y = β₀ + β₁X |
| 다중 선형회귀 | 독립변수 2개 이상, Y = β₀ + β₁X₁ + β₂X₂ + ... |
| 다항 회귀 | 비선형 관계, Y = β₀ + β₁X + β₂X² + ... |
### 로지스틱회귀 유형
| 유형 | 설명 |
|------|------|
| 이진 분류 | 2개 클래스 (Yes/No) |
| 다항 분류 | 3개 이상 클래스 (Softmax) |
| 순서형 분류 | 순서가 있는 범주 (상/중/하) |
### 회귀분석 평가 지표
| 지표 | 설명 | 용도 |
|------|------|------|
| R² (결정계수) | 설명력 (0~1) | 선형회귀 |
| RMSE | 평균 제곱근 오차 | 선형회귀 |
| MAE | 평균 절대 오차 | 선형회귀 |
| 정확도/F1 | 분류 성능 | 로지스틱회귀 |
### 회귀분석 가정
- **선형성**: 독립변수와 종속변수 간 선형 관계
- **독립성**: 잔차 간 상관관계 없음
- **등분산성**: 잔차의 분산이 일정
- **정규성**: 잔차가 정규분포
- **다중공선성 없음**: 독립변수 간 상관관계 낮음
