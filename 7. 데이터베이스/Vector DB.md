#데이터베이스 #AI #벡터 #필수
## 정의
고차원 벡터 저장 및 검색, Vector DB
- 텍스트, 이미지 등을 임베딩 벡터로 변환하여 저장하고 유사도 검색을 수행하는 데이터베이스
- AI/ML 애플리케이션에서 의미 기반 검색(Semantic Search)을 지원
## 키워드
* 임베딩, 유사도 검색, ANN, HNSW, 코사인 유사도, RAG
## 암기법
* "임유인코" - 임베딩, 유사도, 인덱싱, 코사인
## 핵심 개념
| 개념 | 설명 |
|------|------|
| 임베딩 | 데이터를 고차원 벡터로 변환 |
| 유사도 검색 | 벡터 간 거리/유사도 계산 |
| ANN | 근사 최근접 이웃 검색 |
| HNSW | 계층적 탐색 가능 소규모 세계 그래프 |
## 구성도
```
┌─────────────────────────────────────────┐
│              Vector DB                   │
├─────────────────────────────────────────┤
│  [텍스트] → [임베딩모델] → [벡터저장]    │
│  "고양이"    [0.2,0.8,...]   Vector Index │
├─────────────────────────────────────────┤
│  [쿼리벡터] → [ANN검색] → [유사문서]     │
└─────────────────────────────────────────┘
```
## 유사도 측정
| 방법 | 공식 | 특징 |
|------|------|------|
| 코사인 유사도 | cos(θ) = A·B / (|A||B|) | 방향 유사성 |
| 유클리드 거리 | √Σ(ai-bi)² | 절대 거리 |
| 내적 | A·B = Σaibi | 크기 고려 |
