#데이터베이스 #데이터분석 #앙상블 #머신러닝 #필수
## 정의
의사결정나무 앙상블 학습
- 다수의 의사결정나무를 배깅(Bagging) 기법으로 결합하여 예측하는 앙상블 알고리즘
- 특징(Feature) 무작위 선택을 추가하여 트리 간 상관관계를 낮추고 일반화 성능 향상
## 키워드
* Decision Tree, Bagging, Feature Randomness
* 다수결 투표, 평균, 과적합 방지
* 특징 중요도(Feature Importance), OOB(Out-of-Bag)
## 암기법
* "랜포는 배특다" - 배깅 + 특징 무작위 + 다수결
* "나무숲투표" - 의사결정나무 → 숲(Forest) → 투표/평균
## 구성요소/특징/유형
### 랜덤 포레스트 핵심 구성
| 구성요소 | 설명 |
|---------|------|
| Bootstrap Sampling | 원본 데이터에서 복원 추출 |
| Feature Randomness | 각 분할 시 특징 부분집합만 사용 |
| Decision Tree | 개별 학습기 (가지치기 없이 성장) |
| Aggregating | 분류: 다수결, 회귀: 평균 |
### 랜덤 포레스트 특징
- **과적합 방지**: 다수 트리 앙상블로 개별 오류 상쇄
- **특징 중요도**: 변수별 기여도 분석 가능
- **결측값 처리**: 결측값이 있어도 높은 정확도 유지
- **정규화 불필요**: 데이터 스케일링 없이 사용 가능
- **병렬 처리**: 트리별 독립 학습으로 병렬화 용이
### 하이퍼파라미터
| 파라미터 | 설명 | 권장값 |
|---------|------|--------|
| n_estimators | 트리 개수 | 100~500 |
| max_features | 분할 시 고려 특징 수 | sqrt(전체 특징 수) |
| max_depth | 트리 최대 깊이 | None (완전 성장) |
| min_samples_split | 분할 최소 샘플 수 | 2 |
### 랜덤 포레스트 vs XGBoost
| 구분 | 랜덤 포레스트 | XGBoost |
|------|--------------|---------|
| 앙상블 방식 | 배깅 | 부스팅 |
| 학습 방식 | 병렬 | 순차 |
| 과적합 | 낮음 | 중간 |
| 정확도 | 양호 | 우수 |
| 해석성 | 양호 | 낮음 |
| 튜닝 난이도 | 쉬움 | 복잡 |
### 적용 분야
- 금융: 신용 위험 평가, 사기 탐지
- 의료: 질병 진단, 약물 반응 예측
- 마케팅: 고객 세분화, 이탈 예측
