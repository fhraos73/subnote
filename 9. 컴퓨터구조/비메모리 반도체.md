#컴퓨터구조 #반도체 #시스템 #필수 #컴퓨터구조/반도체
## 정의
정보를 처리하는 목적의 반도체, Non-Memory Semiconductor
- 시스템 반도체로 전자제품의 두뇌 역할을 하는 칩으로, 연산/제어/통신 등을 수행하는 반도체
## 키워드
* CPU, GPU, ASIC, FPGA, AP(Application Processor), SoC, NPU
## 암기법
* "시연제통" - 시스템, 연산, 제어, 통신
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| CPU | 범용 연산 프로세서 | Intel, AMD |
| GPU | 그래픽/병렬 연산 | NVIDIA, AMD |
| ASIC | 주문형 반도체 | 특정 용도 최적화 |
| FPGA | 재프로그래밍 가능 | 현장 프로그래밍 |
| SoC | 시스템온칩 | AP 등 통합형 |
