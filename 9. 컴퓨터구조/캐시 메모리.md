#컴퓨터구조 #캐시 #메모리 #필수 #컴퓨터구조/메모리
## 정의
CPU와 주기억장치 속도차 극복을 위한 고속 메모리, Cache Memory
- CPU와 주기억장치 속도차를 극복하기 위해 CPU와 주기억장치 사이에 설치하는 소규모 고속 반도체 기억장치
## 키워드
* 지역성(Locality), 시간적 지역성, 공간적 지역성, 순차적 지역성, 적중률(Hit Ratio), L1/L2/L3
## 암기법
* "시공순" - 시간적, 공간적, 순차적 지역성
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| L1 캐시 | CPU 코어 내장, 최고속 | 수십KB |
| L2 캐시 | 코어별 또는 공유 | 수백KB~수MB |
| L3 캐시 | 전체 코어 공유 | 수MB~수십MB |
| 교체 알고리즘 | LRU, FIFO, LFU | 캐시 관리 |
