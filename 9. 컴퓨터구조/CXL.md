#컴퓨터구조 #메모리 #CXL #인터페이스 #필수 #컴퓨터구조/메모리
## 정의
PCIe 기반 통합 고속 인터페이스 표준, CXL
- PCI 익스프레스(PCIe) 인터페이스 기반으로 CPU 등 여러 장치와 메모리를 연결하는 통합 고속 인터페이스 표준
## 키워드
* PCIe 기반, CXL.io, CXL.cache, CXL.memory, 메모리 풀링, 이기종 컴퓨팅
## 암기법
* "아캐메" - CXL.io(I/O), CXL.cache(캐시), CXL.memory(메모리)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| CXL.io | PCIe 호환 I/O 프로토콜 | 디바이스 디스커버리 |
| CXL.cache | 호스트 메모리 캐싱 | 디바이스→호스트 캐시 |
| CXL.memory | 호스트의 디바이스 메모리 접근 | 메모리 확장 |
