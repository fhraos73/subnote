#컴퓨터구조 #스토리지 #저장장치 #필수 #컴퓨터구조/스토리지
## 정의
데이터 저장 방식에 따른 스토리지 분류, Storage Types
- 디스크를 통한 2차 저장 공간으로 블록/파일/오브젝트 스토리지로 구분
## 키워드
* 블록 스토리지, 파일 스토리지, 오브젝트 스토리지, DAS, NAS, SAN, SDS
## 암기법
* "블파오" - 블록, 파일, 오브젝트
* "다나산" - DAS, NAS, SAN
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| DAS | 직접 연결 스토리지 | SCSI, 소규모 |
| NAS | 네트워크 파일 서버 | 이더넷, 중규모 |
| SAN | 고속 전용 스토리지 네트워크 | FC 스위치, 대규모 |
| SDS | SW 정의 스토리지 | HW 분리 추상화 |
| 블록 | 디스크 블록 단위 접근 | SAN |
| 파일 | 파일 시스템 기반 | NAS |
| 오브젝트 | 메타데이터+데이터 객체 | 클라우드 |
