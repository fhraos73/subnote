#컴퓨터구조 #반도체 #메모리 #필수 #컴퓨터구조/반도체
## 정의
정보를 저장하는 목적의 반도체, Memory Semiconductor
- 정보를 기억하는 목적으로 만들어진 반도체로, 휘발성의 RAM과 비휘발성의 ROM으로 구분
## 키워드
* DRAM, SRAM, ROM, Flash Memory, DDR5, LPDDR5
## 암기법
* "디에스플" - DRAM, SRAM, Flash(NOR/NAND)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| DRAM | 커패시터+트랜지스터, 리프레시 필요 | 주기억장치 |
| SRAM | 플립플롭 기반, 리프레시 불필요 | 캐시 메모리 |
| ROM | 비휘발성, 읽기 전용 | PROM, EPROM, EEPROM |
| Flash | 비휘발성, 블록 소거 | NOR(코드), NAND(데이터) |
