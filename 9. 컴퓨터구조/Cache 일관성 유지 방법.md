#컴퓨터구조 #캐시 #일관성 #필수 #컴퓨터구조/메모리
## 정의
멀티프로세서 환경에서 캐시 데이터 일관성 보장 기법, Cache Coherence
- 캐시 이용 시 지역성(시간적, 공간적, 순차적) 성질을 기반으로 다중 캐시 간 데이터 일관성을 유지하는 방법
## 키워드
* Snooping, Write Update, Write Invalidate, Directory 기반, 버스 감시
## 암기법
* "버갱무 디플리차" - 버스감시(갱신, 무효화), 디렉토리(Full, Limited, Chained)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 공유캐시 | 1개 캐시만 이용 | 단순하나 병목 |
| 공유변수 미저장 | 캐시에 공유변수 저장 안함 | 성능 저하 |
| 버스감시(Snooping) | 버스 모니터링 방식 | 갱신/무효화 |
| Directory | 디렉토리에 저장 유지 | Full/Limited/Chained |
