#컴퓨터구조 #AI #뉴로모픽 #반도체 #필수 #컴퓨터구조/프로세서
## 정의
인간 뇌의 신경 구조를 모사한 차세대 반도체, 뉴로모픽칩
- 사람의 사고과정과 비슷하게 정보를 처리하는 새로운 반도체로 뇌의 작동방식을 최대한 실리콘에 구현한 기술
- 코어가 뉴런 역할, 메모리칩이 시냅스 역할
## 키워드
* 뉴런(Neuron), 시냅스(Synapse), 멤리스터(Memristor), 병렬구성, 저전력, TrueNorth
## 암기법
* "뉴시멤" - 뉴런, 시냅스, 멤리스터
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 코어 | 뉴런 역할, 병렬 구성 | 13만개 뉴런 |
| 메모리 | 시냅스 연결 역할 | 1억3천만개 시냅스 |
| 멤리스터 | 고밀도 시냅스 연결 기술 | 단위면적 고밀도 |
| 동향 | IBM(TrueNorth), 퀄컴(Zeroth) | 국내: 네패스 |
