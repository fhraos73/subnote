#컴퓨터구조 #캐시 #사상 #필수 #컴퓨터구조/메모리
## 정의
캐시 메모리와 주기억장치 간 주소 매핑 방식, Cache Mapping
- 주기억장치의 블록을 캐시 메모리의 슬롯에 매핑하는 3가지 기법
## 키워드
* 직접사상(Direct), 연관사상(Associative), 집합연관사상(Set-Associative), 태그, 슬롯
## 암기법
* "지연집 태라단" - 직접/연관/집합연관, 태그/라인/단어
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 직접사상 | 주소→특정 슬롯 고정 매핑 | 구현 단순, 충돌 多 |
| 연관사상 | 어느 슬롯이든 저장 가능 | 적중률 高, 비용 高 |
| 집합연관사상 | 집합 내 자유 배치 | 직접+연관 절충 |
## 구성도
```
[직접사상]        [연관사상]        [집합연관사상]
주소→슬롯 1:1    주소→전체 탐색    주소→집합→슬롯
태그|슬롯|단어    태그|단어          태그|집합|단어
적중률 낮음       적중률 높음        절충 방식
```
