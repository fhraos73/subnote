#자료구조 #색인 #다차원 #인덱스 #필수 #자료구조/색인
## 정의
비정형 데이터의 효율적 검색을 위한 다차원 색인, Multidimensional Index Structure
- 이미지나 멀티미디어 데이터와 같은 비정형 데이터의 효율적 검색을 위해 여러 개의 필드를 동시에 키(key)로 사용한 색인구조
## 키워드
* R-Tree, k-d Tree, Quad Tree, Grid File, MBR, 공간 데이터, GIS
## 암기법
* "피사케그알쿼" - PAM, SAM, k-d트리, Grid File, R-트리, Quad Tree
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| R-Tree | MBR 기반 공간 데이터 인덱싱 | GIS, LBS |
| k-d Tree | k차원 공간 분할 | 이진 트리 확장 |
| Quad Tree | 2차원 공간 4분할 | 이미지 검색 |
| Grid File | 격자 기반 인덱싱 | 균등 분할 |
| PAM | 점 접근 기법 | 포인트 질의 |
| SAM | 공간 접근 기법 | 영역 질의 |
