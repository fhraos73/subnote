#자료구조 #알고리즘 #복잡도 #필수 #자료구조/알고리즘분석
## 정의
알고리즘 성능을 점근적으로 표현하는 표기법, Big-O Notation
- 데이터 수 N에 대해서 복잡도가 어떤 함수로 나타나는가를 간단히 표현하기 위한 알고리즘 표현법
## 키워드
* 시간복잡도, 공간복잡도, Big-O, Big-Ω, Big-Θ, 최악/평균/최선
## 암기법
* "오오세" - O(상한), Ω(하한), Θ(상하한 교집합)
## 구성요소/특징/유형
| 표기 | 설명 | 비고 |
| ---- | ---- | ---- |
| O(1) | 상수 시간 | 해시 탐색 |
| O(log n) | 로그 시간 | 이진 탐색 |
| O(n) | 선형 시간 | 순차 탐색 |
| O(n log n) | 선형로그 | 병합/퀵 정렬 |
| O(n²) | 이차 시간 | 버블/선택 정렬 |
| O(2ⁿ) | 지수 시간 | 피보나치(재귀) |
