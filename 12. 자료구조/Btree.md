#자료구조 #트리 #B트리 #인덱스 #필수 #자료구조/트리
## 정의
하나의 노드에 여러 데이터를 배치하는 균형 트리, B-Tree
- 하나의 노드에 여러 데이터가 배치되는 트리구조(N개면, N차 트리)
- 삽입과 삭제 시 스스로 균형 유지
## 키워드
* 균형트리, N차 트리, Leaf 노드, 오버플로우, 인덱스, 삽입/삭제 규칙
## 암기법
* "부왼오 리삽오" - 부모기준 왼쪽(작은값)/오른쪽(큰값), Leaf부터 삽입, 오버플로우시 부모로
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| B-Tree | 기본 균형 트리 | N차 노드 |
| B+Tree | 순차검색 향상, Index Set + Sequence Set | 리프 노드 연결 |
| B*Tree | 2/3 채워야 분열 | B+트리 분열 부하 감소 |
| 삽입 규칙 | Leaf부터 삽입, 오버플로우→부모 승격 | 균형 유지 |
| 삭제 규칙 | 언더플로우→재분배/합병 | 균형 유지 |
