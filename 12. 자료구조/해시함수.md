#자료구조 #해시 #탐색 #필수 #자료구조/해시
## 정의
임의 길이 입력을 고정 길이 출력으로 매핑하는 함수, Hash Function
- 임의의 길이의 이진 문자열을 고정된 길이의 이진 문자열로 매핑하는 함수
- 해시 탐색 시 O(1) 시간 복잡도
## 키워드
* 해시 테이블, 충돌(Collision), 오버플로우, 개방주소법, 체이닝, 해시값
## 암기법
* "나폴리는 중세기" - 나눗셈법, 폴딩법, 리해싱, 중간제곱법, 세자릿수분석, 기수변환
* "선재이무/체" - 충돌해결: 선형조사, 재해싱, 이차조사, 무작위 / 체이닝
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 나눗셈법 | h(k) = k mod m | 가장 일반적 |
| 폴딩법 | 키를 분할하여 합산 | 큰 키 값 |
| 중간제곱법 | 키 제곱 후 중간 자릿수 추출 | 균등 분포 |
| 개방주소법 | 빈 슬롯 탐색(선형/이차/이중) | 외부 메모리 불필요 |
| 체이닝 | 같은 주소에 연결 리스트 | 오버플로우 해결 |
