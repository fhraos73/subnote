#자료구조 #트리 #AVL #균형 #필수 #자료구조/트리
## 정의
균형인자 기반 자동 균형 이진 탐색 트리, AVL Tree
- 균형인자(Balance Factor) = 왼쪽트리의 높이 - 오른쪽 트리의 높이
- |BF| ≤ 1을 유지하도록 회전 연산 수행
## 키워드
* 균형인자(BF), Rotation(회전), LL, RR, LR, RL, 이진탐색트리
## 암기법
* "알엘엘알" - LL, RR, LR, RL 4가지 회전
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| LL 회전 | 왼쪽-왼쪽 불균형 → 오른쪽 회전 | 단순 회전 |
| RR 회전 | 오른쪽-오른쪽 불균형 → 왼쪽 회전 | 단순 회전 |
| LR 회전 | 왼쪽-오른쪽 불균형 → 좌회전+우회전 | 이중 회전 |
| RL 회전 | 오른쪽-왼쪽 불균형 → 우회전+좌회전 | 이중 회전 |
| 시간복잡도 | O(log n) | 삽입/삭제/탐색 |
