#자료구조 #해시 #동적해싱 #필수 #자료구조/해시
## 정의
데이터 증가에 따라 동적으로 확장되는 해싱 기법, Extendible Hashing
- 버킷 오버플로우 발생 시 디렉토리를 동적으로 확장하여 재해싱 없이 데이터를 분산 저장하는 기법
## 키워드
* 디렉토리, 버킷, 글로벌 깊이, 로컬 깊이, 동적 확장, 재해싱 불필요
## 암기법
* "디버글로" - 디렉토리, 버킷, 글로벌/로컬 깊이
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 디렉토리 | 해시값→버킷 포인터 배열 | 동적 확장 |
| 버킷 | 실제 데이터 저장 | 고정 크기 |
| 글로벌 깊이 | 디렉토리 사용 비트 수 | 전체 깊이 |
| 로컬 깊이 | 각 버킷의 사용 비트 수 | 개별 깊이 |
| 분할 | 버킷 오버플로우 시 분할 | 글로벌≤로컬 |
