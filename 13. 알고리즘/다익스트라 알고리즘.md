#알고리즘 #그래프 #최단경로 #필수 #알고리즘/그래프
## 정의
한 노드에서 모든 노드까지의 최소 비용 경로를 계산하는 알고리즘, Dijkstra
- 한 노드에서 네트워크 다른 모든 노드에 이르는 최소 비용(least-cost) 경로 계산
- Greedy 기법 기반, 음수 가중치 불가
## 키워드
* Greedy, 최단경로, 서브셋 최적화, 우선순위 큐, 인접 리스트, O(n²), O(n log n)
## 암기법
* "계진" - 계산복잡도(O(n²)), 진동문제(유사 비용 간 진동)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 방법 | Greedy(탐욕) 기법 | 매 단계 최적 선택 |
| 복잡도 | O(n²), 우선순위 큐 시 O(n log n) | 효율적 구현 |
| 제한 | 음수 가중치 불가 | 벨만포드로 해결 |
| 절차 | 시작 노드 선택→인접 노드 비용 갱신→최소 비용 노드 선택→반복 | 단계적 확장 |
## 구성도
```
[시작 노드 S]
    │
    ├── 인접 노드 비용 계산
    │
    ├── 최소 비용 노드 선택 (Greedy)
    │
    ├── 선택 노드의 인접 노드 비용 갱신
    │
    └── 모든 노드 방문까지 반복
```
