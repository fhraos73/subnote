#알고리즘 #그래프 #MST #필수 #알고리즘/그래프
## 정의
가중치 합이 최소인 신장트리, MST(Minimum Spanning Tree)
- 방향성 그래프에서 정점을 연결하는 간선의 가중치 합이 최소가 되는 신장트리
- 사이클 없이 모든 정점을 연결하며 n-1개 간선 포함
## 키워드
* 크루스칼(Kruskal), 프림(Prim), 솔린(Solin), 간선 가중치, 사이클 제거, Union-Find
## 암기법
* "크프솔" - 크루스칼, 프림, 솔린 (3가지 구현 알고리즘)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 크루스칼 | 간선 중심, 가중치 오름차순 선택 | Greedy, Union-Find |
| 프림 | 정점 중심, 인접 최소 간선 선택 | Greedy, 다익스트라 유사 |
| 솔린 | 각 정점에서 최소 간선 동시 선택 | 병렬 처리 가능 |
| 특징 | n개 정점 → n-1개 간선 | 사이클 無 |
## 구성도
```
[크루스칼]              [프림]
간선 정렬(오름차순)     시작 정점 선택
  │                      │
최소 간선 선택          인접 최소 간선 선택
  │                      │
사이클 검사(Union-Find)  트리에 추가
  │                      │
n-1개 선택까지 반복      모든 정점 포함까지 반복
```
