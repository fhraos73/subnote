#운영체제 #메모리관리 #관리정책 #필수 #운영체제/메모리관리
## 정의
가상메모리의 4대 관리정책, Memory Management Policy
- 할당(Allocation), 배치(Placement), 인출(Fetch), 교체(Replacement) 정책
## 키워드
* 할당정책, 배치정책, 인출정책, 교체정책, 연속할당, 불연속할당
## 암기법
* "할배인교" - 할당(메모리 양), 배치(어디에), 인출(시점), 교체(어떤 것)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 할당(연속) | 단일-오버레이/스와핑, 다중-고정/가변분할 | 메모리 양 결정 |
| 할당(불연속) | 페이징, 세그먼트, 페이지드 세그먼트 | 비연속 배치 |
| 배치 | First/Best/Next/Worst Fit | 위치 결정 |
| 인출 | 요구호출(Demand), 예측호출(Anticipatory) | 적재 시점 |
| 교체 | LRU, FIFO, LFU, OPT, MRU | 제거 대상 결정 |
