#운영체제 #교착상태 #동시성제어 #필수 #운영체제/프로세스관리
## 정의
타임스탬프 기반 교착상태 예방 기법, Wait-Die / Wound-Wait
- 트랜잭션의 타임스탬프를 비교하여 교착상태를 예방하는 비선점/선점 기반 프로토콜
## 키워드
* 타임스탬프, 비선점(Wait-Die), 선점(Wound-Wait), 롤백, 재시작
## 암기법
* "늙대기 젊롤 / 늙선점 젊대기" - Wait-Die(늙으면 대기, 젊으면 롤백) / Wound-Wait(늙으면 선점, 젊으면 대기)
## 구성요소/특징/유형
| 구분 | Wait-Die | Wound-Wait |
| ---- | ---- | ---- |
| 방식 | 비선점 기반 | 선점 기반 |
| 오래된 Tx→젊은 Tx | 대기(Wait) | 선점(Wound) |
| 젊은 Tx→오래된 Tx | 롤백(Die) | 대기(Wait) |
| 롤백 빈도 | 젊은 Tx 롤백 많음 | 젊은 Tx 롤백 적음 |
