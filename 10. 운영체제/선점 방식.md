#운영체제 #스케줄링 #선점 #필수 #운영체제/프로세스관리
## 정의
실행 중인 프로세스를 강제로 중단하고 CPU를 재할당하는 스케줄링, Preemptive Scheduling
- 우선순위가 높은 프로세스가 도착하면 현재 실행 중인 프로세스를 중단하고 CPU를 할당
## 키워드
* RR(Round Robin), SRT(Shortest Remaining Time), 다단계 큐, 다단계 피드백 큐
## 암기법
* "라에다다" - RR, SRT(SJF의 선점), 다단계 큐, 다단계 피드백 큐
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| RR(Round Robin) | 시간 할당량(Time Quantum) 기반 순환 | 공정성 보장 |
| SRT | 남은 시간 최소 프로세스 우선 | SJF 선점 버전 |
| 다단계 큐 | 큐별 우선순위 고정 | 큐 간 이동 불가 |
| 다단계 피드백 큐 | 큐 간 이동 가능 | 가장 일반적 |
