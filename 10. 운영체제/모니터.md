#운영체제 #동기화 #모니터 #필수 #운영체제/프로세스관리
## 정의
고수준 동기화 구조체, Monitor
- 공유 데이터와 그 데이터에 대한 연산을 하나의 모듈로 캡슐화한 고수준 동기화 기법
- 세마포어의 복잡성을 해결하기 위해 한센(Hansen)과 호어(Hoare)가 제안
## 키워드
* 상호배제 자동 보장, 조건변수(Condition Variable), wait(), signal(), 캡슐화
## 암기법
* "캡조웨시" - 캡슐화, 조건변수, wait, signal
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 진입 큐 | 모니터 진입 대기 큐 | 상호배제 자동 |
| 조건 변수 | wait/signal 연산 | 동기화 제어 |
| 공유 데이터 | 모니터 내부 데이터 | 외부 접근 차단 |
| 프로시저 | 공유 데이터 접근 함수 | 한 번에 하나만 |
