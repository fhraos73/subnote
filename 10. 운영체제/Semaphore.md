#운영체제 #동기화 #세마포어 #필수 #운영체제/프로세스관리
## 정의
프로세스 간 동기화를 위한 정수형 변수 기반 메커니즘, Semaphore
- 다익스트라가 제안한 P(wait)/V(signal) 연산을 통해 임계영역 접근을 제어하는 동기화 도구
## 키워드
* P연산(wait), V연산(signal), 이진 세마포어, 카운팅 세마포어, 임계영역, 상호배제
## 암기법
* "PV" - P(Proberen=검사=wait), V(Verhogen=증가=signal)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 이진 세마포어 | 0 또는 1 값 | 뮤텍스와 유사 |
| 카운팅 세마포어 | 0~N 값 | 다중 자원 제어 |
| P연산(wait) | S--; S<0이면 블록 | 자원 획득 시도 |
| V연산(signal) | S++; S≤0이면 깨움 | 자원 반환 |
