#운영체제 #스케줄링 #CPU #필수 #운영체제/프로세스관리
## 정의
CPU 자원을 프로세스에 효율적으로 배분하는 기법, CPU Scheduling
- 다중 프로그래밍 환경에서 준비 큐의 프로세스에게 CPU를 할당하는 정책과 알고리즘
## 키워드
* 선점, 비선점, 디스패처, 스케줄링 큐, 응답시간, 처리량, 대기시간, 반환시간
## 암기법
* "선비" - 선점(Preemptive), 비선점(Non-preemptive) 방식
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 장기 스케줄러 | 디스크→메모리 적재 | Job Scheduler |
| 중기 스케줄러 | 메모리 관리(스와핑) | Swapper |
| 단기 스케줄러 | 준비→실행 CPU 할당 | CPU Scheduler |
