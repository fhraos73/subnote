#운영체제 #인터럽트 #하드웨어 #필수 #운영체제/프로세스관리
## 정의
CPU에 비동기적으로 발생하는 처리 요청 신호, Interrupt
- 프로그램 실행 중 예기치 않은 상황이 발생하거나 외부 장치의 서비스 요청 시 CPU에 알리는 메커니즘
## 키워드
* 외부 인터럽트, 내부 인터럽트, 소프트웨어 인터럽트, ISR, 인터럽트 벡터, 마스킹
## 암기법
* "외내소" - 외부(HW), 내부(예외), 소프트웨어(SVC/Trap)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 외부 인터럽트 | 전원이상, 타이머, I/O | HW 신호 |
| 내부 인터럽트 | 0 나누기, 오버플로우, 보호위반 | 예외(Exception) |
| SW 인터럽트 | SVC(System Call), Trap | 의도적 발생 |
| ISR | 인터럽트 서비스 루틴 | 핸들러 실행 |
| 인터럽트 벡터 | ISR 주소 테이블 | 빠른 분기 |
## 구성도
```
[실행 중] ──인터럽트 발생──▶ [현재 상태 저장(PCB)]
                                    │
                              [인터럽트 벡터 참조]
                                    │
                              [ISR 실행]
                                    │
                              [상태 복원] ──▶ [실행 재개]
```
