#운영체제 #프로세스 #문맥교환 #필수 #운영체제/프로세스관리
## 정의
실행 중인 프로세스 상태를 저장하고 다른 프로세스 상태를 복원하는 작업, Context Switching
- CPU가 현재 프로세스의 상태(PCB)를 저장하고, 새로운 프로세스의 상태를 적재하여 실행을 전환하는 과정
## 키워드
* PCB(Process Control Block), 레지스터 저장/복원, 오버헤드, 디스패처, 스케줄러
## 암기법
* "저적전" - 현재 상태 저장, 새 상태 적재, 실행 전환
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| PCB 저장 | 현재 프로세스 레지스터/PC 저장 | 커널 메모리 |
| PCB 복원 | 새 프로세스 레지스터/PC 복원 | 실행 재개 |
| 오버헤드 | 문맥교환에 소요되는 시간 | 최소화 중요 |
| 발생 원인 | 인터럽트, 시스템콜, 스케줄링 | 비자발/자발적 |
