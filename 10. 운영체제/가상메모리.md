#운영체제 #메모리관리 #가상메모리 #필수 #운영체제/메모리관리
## 정의
사용자에게 실제보다 큰 메모리 공간을 제공하는 기법, Virtual Memory
- 사용자에게 주기억장치보다 더 큰 용량의 가상 기억 공간을 제공하는 기억장치 관리기법
- 구조: CODE, DATA, BSS, HEAP, STACK
## 키워드
* 가상주소, 물리주소, 페이지 테이블, MMU, TLB, 요구 페이징, 스와핑
## 암기법
* "할배인교" - 할당, 배치, 인출, 교체 (4대 관리정책)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 할당 | 연속(단일/다중), 불연속(페이징/세그먼트) | 메모리 양 결정 |
| 배치 | First/Best/Next/Worst Fit | 어디에 배치 |
| 인출 | 요구호출, 예측호출 | 언제 적재 |
| 교체 | LRU, FIFO, LFU, OPT | 어떤것 교체 |
