#운영체제 #스케줄링 #비선점 #필수 #운영체제/프로세스관리
## 정의
프로세스가 자발적으로 CPU를 반환할 때까지 실행을 보장하는 스케줄링, Non-preemptive Scheduling
- 실행 중인 프로세스가 완료되거나 I/O 요청 시까지 CPU를 점유
## 키워드
* FCFS, SJF, HRN, Priority, 호위효과(Convoy Effect)
## 암기법
* "퍼숏에프" - FCFS, SJF, HRN, Priority
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| FCFS | 먼저 도착한 순서대로 | 호위효과 발생 |
| SJF | 실행시간 짧은 순 | 기아 현상 가능 |
| HRN | (대기시간+서비스시간)/서비스시간 | 기아 해결 |
| Priority | 우선순위 기반 | 에이징으로 기아 해결 |
