#운영체제 #메모리관리 #페이징 #필수 #운영체제/메모리관리
## 정의
고정 크기 페이지 단위 메모리 분할 기법, Paging
- 페이지 테이블을 이용하여 가상주소를 물리주소로 변환하는 고정 크기 분할 방식
## 키워드
* 페이지, 프레임, 페이지 테이블, 페이지 폴트, TLB, 내부 단편화
## 암기법
* "페프테" - 페이지(논리), 프레임(물리), 테이블(매핑)
## 구성요소/특징/유형
| 구분 | 설명 | 비고 |
| ---- | ---- | ---- |
| 페이지 | 가상메모리의 고정 크기 블록 | 논리적 단위 |
| 프레임 | 물리메모리의 고정 크기 블록 | 물리적 단위 |
| 페이지 테이블 | 페이지→프레임 매핑 | MMU 관리 |
| TLB | 페이지 테이블 캐시 | 고속 주소변환 |
| 페이지 폴트 | 페이지 미적재 시 인터럽트 | 디스크→메모리 |
